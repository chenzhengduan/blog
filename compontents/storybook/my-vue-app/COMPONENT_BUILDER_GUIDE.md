# 动态组件配置器使用说明

## 项目概述

这是一个动态组件配置系统，允许用户在运行时动态添加、配置和管理各种表单组件。

## 功能特性

### 1. 支持的组件类型

- **输入框 (input)**: 普通文本输入
- **数字输入框 (input-number)**: 带数字控制的输入框，支持最小值、最大值、步长
- **下拉选择 (select)**: 自定义选项的下拉选择框
- **日期选择 (date)**: 日期选择器
- **日期时间 (datetime)**: 日期时间选择器
- **时间选择 (time)**: 时间选择器
- **开关 (switch)**: 开关切换组件
- **滑块 (slider)**: 滑块选择器，支持最小值和最大值
- **文本域 (textarea)**: 多行文本输入

### 2. 核心功能

#### 添加组件
1. 点击右上角"添加组件"按钮
2. 选择组件类型
3. 填写配置信息：
   - 标签名称：显示在左侧的标签文本
   - 字段名：数据字段的唯一标识（英文）
   - 占位符：组件的提示文本
   - 默认值：组件的初始值

#### 特殊配置

**下拉选择 (select)**
- 可自定义选项列表
- 每个选项包含：
  - 选项名称：显示给用户的文本
  - 选项值：实际存储的值
- 支持动态添加/删除选项

**数字输入框 (input-number)**
- 最小值：允许输入的最小数字
- 最大值：允许输入的最大数字
- 步长：点击增减按钮时的变化幅度

**滑块 (slider)**
- 最小值：滑块的最小值
- 最大值：滑块的最大值

#### 编辑组件
- 点击组件右侧的"编辑"按钮
- 修改配置后点击"确定"保存

#### 删除组件
- 点击组件右侧的"删除"按钮
- 组件将从列表中移除

#### 数据导出

**导出配置**
- 导出所有组件的配置信息
- 包含组件类型、标签、字段名等完整配置
- 配置数据会复制到剪贴板，可用于保存或分享

**导出值**
- 导出所有组件当前的值
- 以 `{ field: value }` 格式导出
- 可用于表单数据提交

## 使用场景

### 场景1：动态表单生成
适用于需要根据不同业务场景动态生成表单的系统。

### 场景2：配置化页面
通过配置而非硬编码的方式构建页面组件。

### 场景3：低代码平台
作为低代码平台的表单设计器组件。

### 场景4：数据收集
灵活配置各种数据收集表单。

## 技术实现

### 组件架构
```
ComponentBuilder.vue (主组件)
├── 组件列表展示
├── 添加/编辑对话框
├── 动态组件渲染 (component :is)
└── 数据预览
```

### 核心技术
- **Vue 3 Composition API**: 响应式数据管理
- **Element Plus**: UI组件库
- **动态组件**: 使用 `<component :is>` 实现动态渲染
- **TypeScript**: 类型安全

### 数据结构

```typescript
interface ComponentConfig {
  id: string              // 组件唯一ID
  type: string           // 组件类型
  label: string          // 标签名称
  field: string          // 字段名
  placeholder?: string   // 占位符
  value: any            // 当前值
  defaultValue?: any    // 默认值
  options?: Array<{     // 下拉选项（仅select）
    label: string
    value: string | number
  }>
  min?: number          // 最小值（数字输入/滑块）
  max?: number          // 最大值（数字输入/滑块）
  step?: number         // 步长（数字输入）
}
```

## 快速开始

### 示例1：创建简单表单

1. 添加"姓名"输入框
   - 类型：输入框
   - 标签：姓名
   - 字段：name
   - 占位符：请输入姓名

2. 添加"年龄"数字输入框
   - 类型：数字输入框
   - 标签：年龄
   - 字段：age
   - 最小值：0
   - 最大值：150

3. 添加"性别"下拉选择
   - 类型：下拉选择
   - 标签：性别
   - 字段：gender
   - 选项：
     - 男 (male)
     - 女 (female)

### 示例2：导出和使用数据

1. 配置好组件后，填写各字段值
2. 点击"导出值"按钮
3. 得到类似如下的数据：
```json
{
  "name": "张三",
  "age": 25,
  "gender": "male"
}
```

## 扩展功能

### 自定义组件类型
可以在 `getComponentType` 函数中添加新的组件类型映射。

### 持久化存储
可以将配置保存到数据库或localStorage，实现配置的保存和加载。

### 表单验证
可以为每个组件添加验证规则，实现表单验证功能。

### 条件显示
可以添加组件的显示条件，实现动态表单逻辑。

## 注意事项

1. **字段名唯一性**: 确保每个组件的字段名（field）唯一
2. **下拉选项**: 下拉选择至少需要一个选项
3. **数值范围**: 数字输入框和滑块要合理设置最小值和最大值
4. **性能考虑**: 大量组件时注意性能优化
