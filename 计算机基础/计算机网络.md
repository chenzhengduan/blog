## 1. 网络基础概念

### 1.1 什么是计算机网络

计算机网络是指将地理位置不同的具有独立功能的多台计算机及其外部设备，通过通信线路连接起来，在网络操作系统、网络管理软件及网络通信协议的管理和协调下，实现资源共享和信息传递的计算机系统。

### 1.2 网络的功能

1. **资源共享**：硬件资源、软件资源、数据资源
2. **数据通信**：信息传递、远程通信
3. **分布式处理**：负载均衡、并行计算
4. **提高可靠性**：备份、冗余
5. **负载均衡**：分散处理任务

### 1.3 网络的分类

#### 1.3.1 按覆盖范围分类

- **局域网（LAN）**：覆盖范围小，通常在几公里以内
- **城域网（MAN）**：覆盖一个城市范围
- **广域网（WAN）**：覆盖范围大，可跨越国家和大洲
- **个人区域网（PAN）**：个人设备间的短距离网络

#### 1.3.2 按传输技术分类

- **广播式网络**：所有节点共享通信信道
- **点对点网络**：每条线路连接一对节点

#### 1.3.3 按拓扑结构分类

- **总线型**：所有节点连接到一条公共总线
- **星型**：所有节点连接到中央节点
- **环型**：节点连接成环状
- **网状型**：节点之间有多条连接路径

### 1.4 网络性能指标

#### 1.4.1 带宽（Bandwidth）

带宽是指网络中某个信道能够传输数据的最大速率，通常以bps（bits per second）为单位。

#### 1.4.2 延迟（Latency）

延迟是指数据从源点到目的点所需要的时间，包括：
- **传播延迟**：信号在传输媒体中传播所需时间
- **传输延迟**：将数据推送到传输媒体所需时间
- **处理延迟**：节点处理数据所需时间
- **排队延迟**：数据在缓冲区等待的时间

#### 1.4.3 吞吐量（Throughput）

吞吐量是指在单位时间内通过某个网络的数据量，是网络实际传输能力的度量。

#### 1.4.4 丢包率（Packet Loss Rate）

丢包率是指在数据传输过程中丢失的数据包占总数据包的比例。

## 2. 网络体系结构

### 2.1 OSI七层模型

OSI（Open Systems Interconnection）模型是国际标准化组织制定的网络通信标准模型。

#### 2.1.1 七层结构

1. **物理层（Physical Layer）**
   - 功能：传输原始比特流
   - 设备：集线器、中继器
   - 协议：RS-232、RJ-45

2. **数据链路层（Data Link Layer）**
   - 功能：提供可靠的点对点数据传输
   - 设备：交换机、网桥
   - 协议：Ethernet、PPP、HDLC

3. **网络层（Network Layer）**
   - 功能：路径选择和逻辑寻址
   - 设备：路由器
   - 协议：IP、ICMP、ARP

4. **传输层（Transport Layer）**
   - 功能：提供端到端的可靠数据传输
   - 协议：TCP、UDP

5. **会话层（Session Layer）**
   - 功能：建立、管理和终止会话
   - 协议：NetBIOS、RPC

6. **表示层（Presentation Layer）**
   - 功能：数据格式转换、加密解密
   - 协议：SSL、TLS、JPEG、MPEG

7. **应用层（Application Layer）**
   - 功能：为应用程序提供网络服务
   - 协议：HTTP、FTP、SMTP、DNS

### 2.2 TCP/IP模型

TCP/IP模型是互联网实际使用的网络模型，包含四层：

1. **网络接口层**：对应OSI的物理层和数据链路层
2. **网际层**：对应OSI的网络层
3. **传输层**：对应OSI的传输层
4. **应用层**：对应OSI的会话层、表示层和应用层

### 2.3 协议栈的工作原理

数据在发送时从应用层向下逐层封装，在接收时从物理层向上逐层解封装。

```
发送端：
应用层数据 → 传输层头部+数据 → 网络层头部+数据 → 数据链路层头部+数据+尾部 → 物理层比特流

接收端：
物理层比特流 → 数据链路层去头尾 → 网络层去头部 → 传输层去头部 → 应用层数据
```

## 3. 物理层

### 3.1 物理层的功能

物理层负责在物理媒体上传输原始的比特流，主要功能包括：

1. **定义物理接口**：连接器的形状、引脚数量等
2. **定义电气特性**：电压范围、阻抗匹配等
3. **定义功能特性**：各引脚的功能
4. **定义过程特性**：建立连接的步骤

### 3.2 传输媒体

#### 3.2.1 有线传输媒体

**双绞线（Twisted Pair）**
- 结构：两根绝缘铜线相互缠绕
- 类型：屏蔽双绞线（STP）、非屏蔽双绞线（UTP）
- 应用：局域网、电话网络

**同轴电缆（Coaxial Cable）**
- 结构：内导体、绝缘层、外导体、保护层
- 特点：抗干扰能力强、传输距离远
- 应用：有线电视、早期以太网

**光纤（Optical Fiber）**
- 结构：纤芯、包层、保护层
- 类型：单模光纤、多模光纤
- 优点：传输距离远、带宽大、抗干扰

#### 3.2.2 无线传输媒体

**无线电波**
- 频段：VLF、LF、MF、HF、VHF、UHF、SHF、EHF
- 特点：传播距离远、穿透能力强
- 应用：广播、移动通信

**微波**
- 特点：直线传播、频率高
- 类型：地面微波、卫星微波
- 应用：长距离通信、卫星通信

**红外线**
- 特点：不能穿透障碍物
- 应用：短距离通信、遥控器

### 3.3 数字信号与模拟信号

#### 3.3.1 信号类型

**数字信号**
- 特点：离散的、有限的取值
- 优点：抗干扰能力强、易于处理
- 缺点：需要更大的带宽

**模拟信号**
- 特点：连续变化的信号
- 优点：带宽利用率高
- 缺点：容易受干扰

#### 3.3.2 编码技术

**数字数据编码为数字信号**
- 不归零编码（NRZ）
- 曼彻斯特编码
- 差分曼彻斯特编码

**数字数据编码为模拟信号**
- 幅移键控（ASK）
- 频移键控（FSK）
- 相移键控（PSK）

### 3.4 多路复用技术

#### 3.4.1 频分多路复用（FDM）

将可用带宽分成多个频段，每个用户使用不同的频段。

#### 3.4.2 时分多路复用（TDM）

将时间分成多个时隙，每个用户在分配的时隙内传输数据。

#### 3.4.3 波分多路复用（WDM）

在光纤中使用不同波长的光信号同时传输多路数据。

#### 3.4.4 码分多路复用（CDM）

每个用户使用不同的编码序列，可以同时使用相同的频段和时间。

## 4. 数据链路层

### 4.1 数据链路层的功能

数据链路层负责在两个相邻节点间提供可靠的数据传输服务。

#### 4.1.1 主要功能

1. **成帧**：将网络层的数据封装成帧
2. **错误检测和纠正**：检测并纠正传输错误
3. **流量控制**：控制发送速率
4. **访问控制**：控制对共享信道的访问

### 4.2 成帧

#### 4.2.1 帧的结构

```
+----------+--------+----------+--------+----------+
| 帧开始符 |  头部  |   数据   |  校验  | 帧结束符 |
+----------+--------+----------+--------+----------+
```

#### 4.2.2 成帧方法

**字符计数法**
- 在帧头指明帧中字符的数量
- 缺点：计数字段出错会导致失步

**字符填充法**
- 使用特殊字符作为帧的开始和结束标志
- 在数据中出现标志字符时进行填充

**比特填充法**
- 使用特定的比特模式作为标志
- 在数据中出现标志模式时插入额外比特

**物理层编码违例法**
- 使用物理层的编码违例作为帧边界

### 4.3 错误检测和纠正

#### 4.3.1 错误类型

**单比特错误**
- 只有一个比特发生错误
- 通常由热噪声引起

**突发错误**
- 连续多个比特发生错误
- 通常由脉冲噪声引起

#### 4.3.2 错误检测方法

**奇偶校验**
```c
// 奇偶校验示例
int calculateParity(int data) {
    int parity = 0;
    while (data) {
        parity ^= (data & 1);
        data >>= 1;
    }
```

#### 6.3.5 TCP流量控制

流量控制防止发送方发送数据过快，导致接收方缓冲区溢出。

```c
// 滑动窗口流量控制
typedef struct {
    uint8_t* buffer;
    int bufferSize;
    int dataStart;      // 数据开始位置
    int dataEnd;        // 数据结束位置
    int windowStart;    // 窗口开始位置
    int windowSize;     // 窗口大小
} ReceiveWindow;

// 初始化接收窗口
void initReceiveWindow(ReceiveWindow* window, int size) {
    window->buffer = malloc(size);
    window->bufferSize = size;
    window->dataStart = 0;
    window->dataEnd = 0;
    window->windowStart = 0;
    window->windowSize = size;
}

// 接收数据到窗口
int receiveToWindow(ReceiveWindow* window, uint8_t* data, int dataLen, uint32_t seqNum) {
    // 检查序列号是否在窗口范围内
    if (seqNum < window->windowStart || 
        seqNum >= window->windowStart + window->windowSize) {
        return -1; // 超出窗口范围
    }
    
    // 计算在缓冲区中的位置
    int bufferPos = (seqNum - window->windowStart + window->dataStart) % window->bufferSize;
    
    // 复制数据到缓冲区
    for (int i = 0; i < dataLen && bufferPos < window->bufferSize; i++) {
        window->buffer[bufferPos] = data[i];
        bufferPos = (bufferPos + 1) % window->bufferSize;
    }
    
    // 更新数据结束位置
    if (seqNum + dataLen > window->dataEnd) {
        window->dataEnd = seqNum + dataLen;
    }
    
    return dataLen;
}

// 从窗口读取数据
int readFromWindow(ReceiveWindow* window, uint8_t* buffer, int maxLen) {
    int bytesRead = 0;
    
    while (bytesRead < maxLen && window->dataStart < window->dataEnd) {
        buffer[bytesRead] = window->buffer[window->dataStart % window->bufferSize];
        window->dataStart++;
        bytesRead++;
    }
    
    // 更新窗口位置
    window->windowStart = window->dataStart;
    
    return bytesRead;
}

// 计算可用窗口大小
int getAvailableWindowSize(ReceiveWindow* window) {
    int usedSpace = window->dataEnd - window->dataStart;
    return window->windowSize - usedSpace;
}
```

#### 6.3.6 TCP拥塞控制

TCP使用多种算法来避免网络拥塞：

```c
// 拥塞控制状态
typedef enum {
    SLOW_START,
    CONGESTION_AVOIDANCE,
    FAST_RECOVERY
} CongestionState;

// 拥塞控制块
typedef struct {
    CongestionState state;
    uint32_t cwnd;          // 拥塞窗口
    uint32_t ssthresh;      // 慢启动阈值
    uint32_t dupAckCount;   // 重复ACK计数
    uint32_t lastAck;       // 最后收到的ACK
} CongestionControl;

// 初始化拥塞控制
void initCongestionControl(CongestionControl* cc, uint32_t mss) {
    cc->state = SLOW_START;
    cc->cwnd = mss;                    // 初始拥塞窗口为1个MSS
    cc->ssthresh = 65535;              // 初始阈值
    cc->dupAckCount = 0;
    cc->lastAck = 0;
}

// 收到新ACK时的处理
void onNewAck(CongestionControl* cc, uint32_t ack, uint32_t mss) {
    cc->lastAck = ack;
    cc->dupAckCount = 0;
    
    switch (cc->state) {
        case SLOW_START:
            // 慢启动：每收到一个ACK，cwnd增加1个MSS
            cc->cwnd += mss;
            
            // 检查是否达到阈值
            if (cc->cwnd >= cc->ssthresh) {
                cc->state = CONGESTION_AVOIDANCE;
                printf("进入拥塞避免阶段，cwnd=%u\n", cc->cwnd);
            }
            break;
            
        case CONGESTION_AVOIDANCE:
            // 拥塞避免：每个RTT增加1个MSS
            cc->cwnd += (mss * mss) / cc->cwnd;
            break;
            
        case FAST_RECOVERY:
            // 快速恢复：收到新ACK，退出快速恢复
            cc->cwnd = cc->ssthresh;
            cc->state = CONGESTION_AVOIDANCE;
            printf("退出快速恢复，cwnd=%u\n", cc->cwnd);
            break;
    }
}

// 收到重复ACK时的处理
void onDuplicateAck(CongestionControl* cc, uint32_t ack, uint32_t mss) {
    if (ack == cc->lastAck) {
        cc->dupAckCount++;
        
        if (cc->dupAckCount == 3) {
            // 快速重传
            printf("检测到3个重复ACK，执行快速重传\n");
            
            // 快速恢复
            cc->ssthresh = max(cc->cwnd / 2, 2 * mss);
            cc->cwnd = cc->ssthresh + 3 * mss;
            cc->state = FAST_RECOVERY;
            
            printf("进入快速恢复，ssthresh=%u, cwnd=%u\n", 
                   cc->ssthresh, cc->cwnd);
        } else if (cc->state == FAST_RECOVERY) {
            // 快速恢复期间的重复ACK
            cc->cwnd += mss;
        }
    }
}

// 超时重传时的处理
void onTimeout(CongestionControl* cc, uint32_t mss) {
    printf("发生超时，执行拥塞控制\n");
    
    // 设置阈值为当前窗口的一半
    cc->ssthresh = max(cc->cwnd / 2, 2 * mss);
    
    // 重新开始慢启动
    cc->cwnd = mss;
    cc->state = SLOW_START;
    cc->dupAckCount = 0;
    
    printf("重新慢启动，ssthresh=%u, cwnd=%u\n", cc->ssthresh, cc->cwnd);
}
```

#### 6.3.7 TCP编程示例

```c
// TCP服务器
void tcpServer(int port) {
    int serverSock = socket(AF_INET, SOCK_STREAM, 0);
    if (serverSock < 0) {
        perror("socket");
        return;
    }
    
    // 设置地址重用
    int opt = 1;
    setsockopt(serverSock, SOL_SOCKET, SO_REUSEADDR, &opt, sizeof(opt));
    
    struct sockaddr_in serverAddr;
    serverAddr.sin_family = AF_INET;
    serverAddr.sin_addr.s_addr = INADDR_ANY;
    serverAddr.sin_port = htons(port);
    
    if (bind(serverSock, (struct sockaddr*)&serverAddr, sizeof(serverAddr)) < 0) {
        perror("bind");
        close(serverSock);
        return;
    }
    
    if (listen(serverSock, 5) < 0) {
        perror("listen");
        close(serverSock);
        return;
    }
    
    printf("TCP服务器启动，监听端口 %d\n", port);
    
    while (1) {
        struct sockaddr_in clientAddr;
        socklen_t clientLen = sizeof(clientAddr);
        
        int clientSock = accept(serverSock, (struct sockaddr*)&clientAddr, &clientLen);
        if (clientSock < 0) {
            perror("accept");
            continue;
        }
        
        printf("客户端连接: %s:%d\n", 
               inet_ntoa(clientAddr.sin_addr), ntohs(clientAddr.sin_port));
        
        // 创建线程处理客户端
        pthread_t thread;
        int* sockPtr = malloc(sizeof(int));
        *sockPtr = clientSock;
        pthread_create(&thread, NULL, handleClient, sockPtr);
        pthread_detach(thread);
    }
    
    close(serverSock);
}

// 客户端处理函数
void* handleClient(void* arg) {
    int clientSock = *(int*)arg;
    free(arg);
    
    char buffer[1024];
    ssize_t bytes;
    
    while ((bytes = recv(clientSock, buffer, sizeof(buffer) - 1, 0)) > 0) {
        buffer[bytes] = '\0';
        printf("收到消息: %s\n", buffer);
        
        // 回显消息
        char response[1024];
        snprintf(response, sizeof(response), "服务器收到: %s", buffer);
        send(clientSock, response, strlen(response), 0);
    }
    
    printf("客户端断开连接\n");
    close(clientSock);
    return NULL;
}

// TCP客户端
void tcpClient(const char* serverIP, int serverPort) {
    int sockfd = socket(AF_INET, SOCK_STREAM, 0);
    if (sockfd < 0) {
        perror("socket");
        return;
    }
    
    struct sockaddr_in serverAddr;
    serverAddr.sin_family = AF_INET;
    inet_pton(AF_INET, serverIP, &serverAddr.sin_addr);
    serverAddr.sin_port = htons(serverPort);
    
    if (connect(sockfd, (struct sockaddr*)&serverAddr, sizeof(serverAddr)) < 0) {
        perror("connect");
        close(sockfd);
        return;
    }
    
    printf("连接到服务器 %s:%d\n", serverIP, serverPort);
    
    char message[1024];
    char response[1024];
    
    while (1) {
        printf("请输入消息 (输入 'quit' 退出): ");
        fgets(message, sizeof(message), stdin);
        
        message[strcspn(message, "\n")] = '\0';
        
        if (strcmp(message, "quit") == 0) {
            break;
        }
        
        // 发送消息
        if (send(sockfd, message, strlen(message), 0) < 0) {
            perror("send");
            break;
        }
        
        // 接收回复
        ssize_t bytes = recv(sockfd, response, sizeof(response) - 1, 0);
        if (bytes > 0) {
            response[bytes] = '\0';
            printf("服务器回复: %s\n", response);
        } else if (bytes == 0) {
            printf("服务器关闭连接\n");
            break;
        } else {
            perror("recv");
            break;
        }
    }
    
    close(sockfd);
 }
```

## 7. 应用层

应用层为用户提供网络服务，包含各种应用协议。

### 7.1 HTTP协议

HTTP（HyperText Transfer Protocol）是万维网的基础协议。

#### 7.1.1 HTTP请求格式

```
GET /index.html HTTP/1.1
Host: www.example.com
User-Agent: Mozilla/5.0
Accept: text/html,application/xhtml+xml
Connection: keep-alive

[请求体]
```

#### 7.1.2 HTTP响应格式

```
HTTP/1.1 200 OK
Content-Type: text/html
Content-Length: 1234
Server: Apache/2.4.41

[响应体]
```

```c
    }
    
    if (listen(serverSock, 10) < 0) {
        perror("listen");
        close(serverSock);
        return;
    }
    
    printf("HTTP服务器启动，监听端口 %d\n", port);
    
    while (1) {
        struct sockaddr_in clientAddr;
        socklen_t clientLen = sizeof(clientAddr);
        
        int clientSock = accept(serverSock, (struct sockaddr*)&clientAddr, &clientLen);
        if (clientSock < 0) {
            perror("accept");
            continue;
        }
        
        // 创建线程处理HTTP请求
        pthread_t thread;
        int* sockPtr = malloc(sizeof(int));
        *sockPtr = clientSock;
        pthread_create(&thread, NULL, handleHttpRequest, sockPtr);
        pthread_detach(thread);
    }
    
    close(serverSock);
}
```

```c
// DNS查询
char* dnsQuery(const char* hostname, const char* dnsServer) {
    int sockfd = socket(AF_INET, SOCK_DGRAM, 0);
    if (sockfd < 0) {
        return NULL;
    }
    
    struct sockaddr_in serverAddr;
    serverAddr.sin_family = AF_INET;
    inet_pton(AF_INET, dnsServer, &serverAddr.sin_addr);
    serverAddr.sin_port = htons(53);

    uint8_t query[512];
    int queryLen = 0;
    
    // DNS头部
    DNSHeader* header = (DNSHeader*)query;
    header->id = htons(rand());
    header->flags = htons(0x0100); // 标准查询
    header->qdcount = htons(1);
    header->ancount = 0;
    header->nscount = 0;
    header->arcount = 0;
    queryLen += sizeof(DNSHeader);
    
    // 编码域名
    char* domain = strdup(hostname);
    char* token = strtok(domain, ".");
    
    while (token != NULL) {
        int len = strlen(token);
        query[queryLen++] = len;
        memcpy(query + queryLen, token, len);
        queryLen += len;
        token = strtok(NULL, ".");
    }
    query[queryLen++] = 0; // 结束标记
    
    // 查询类型和类
    *(uint16_t*)(query + queryLen) = htons(1); // A记录
    queryLen += 2;
    *(uint16_t*)(query + queryLen) = htons(1); // IN类
    queryLen += 2;
    
    free(domain);
    
    // 发送查询
    sendto(sockfd, query, queryLen, 0, 
           (struct sockaddr*)&serverAddr, sizeof(serverAddr));
    
    // 接收响应
    uint8_t response[512];
    ssize_t responseLen = recvfrom(sockfd, response, sizeof(response), 0, NULL, NULL);
    
    if (responseLen > 0) {
        // 解析响应
        DNSHeader* respHeader = (DNSHeader*)response;
        
        if (ntohs(respHeader->ancount) > 0) {
            // 跳过查询部分
            int offset = sizeof(DNSHeader);
            
            // 跳过域名
            while (response[offset] != 0) {
                offset += response[offset] + 1;
            }
            offset += 5; // 结束标记 + 类型 + 类
            
            // 解析回答
            offset += 2; // 跳过名称指针
            uint16_t type = ntohs(*(uint16_t*)(response + offset));
            offset += 8; // 跳过类型、类、TTL
            uint16_t dataLen = ntohs(*(uint16_t*)(response + offset));
            offset += 2;
            
            if (type == 1 && dataLen == 4) { // A记录
                static char ipStr[16];
                snprintf(ipStr, sizeof(ipStr), "%d.%d.%d.%d",
                        response[offset], response[offset+1],
                        response[offset+2], response[offset+3]);
                close(sockfd);
                return ipStr;
            }
        }
    }
    
    close(sockfd);
    return NULL;
}
```

```c
// 模幂运算
uint64_t modPow(uint64_t base, uint64_t exp, uint64_t mod) {
    uint64_t result = 1;
    base = base % mod;
    
    while (exp > 0) {
        if (exp % 2 == 1) {
            result = (result * base) % mod;
        }
        exp = exp >> 1;
        base = (base * base) % mod;
    }
    
    return result;
}
```

```c
typedef struct {

    uint64_t signature;    // RSA签名
} DigitalSignature;

// 创建数字签名
void createDigitalSignature(uint8_t* message, int messageLen, 
                           RSAKeyPair* keyPair, DigitalSignature* sig) {
    // 计算消息哈希
    sha256(message, messageLen, sig->hash);
    
    // 将哈希转换为整数（简化）
    uint64_t hashInt = 0;
    for (int i = 0; i < 8; i++) {
        hashInt = (hashInt << 8) | sig->hash[i];
    }
    
    // 使用私钥签名
    sig->signature = rsaEncrypt(hashInt, keyPair->d, keyPair->n);
}

// 验证数字签名
bool verifyDigitalSignature(uint8_t* message, int messageLen,
                           DigitalSignature* sig, uint64_t publicKey, uint64_t n) {
    // 计算消息哈希
    uint8_t computedHash[32];
    sha256(message, messageLen, computedHash);
    
    // 使用公钥解密签名
    uint64_t decryptedHash = rsaDecrypt(sig->signature, publicKey, n);
    
    // 将计算的哈希转换为整数
    uint64_t hashInt = 0;
    for (int i = 0; i < 8; i++) {
        hashInt = (hashInt << 8) | computedHash[i];
    }
    
    return decryptedHash == hashInt;
 }
```

```c
TokenBucket* initTokenBucket(int capacity, int rate) {
    TokenBucket* bucket = malloc(sizeof(TokenBucket));
    bucket->capacity = capacity;
    bucket->tokens = capacity;
    bucket->rate = rate;
    bucket->lastUpdate = time(NULL);

// 更新令牌桶
void updateTokenBucket(TokenBucket* bucket) {
    time_t now = time(NULL);
    time_t elapsed = now - bucket->lastUpdate;
    
    if (elapsed > 0) {
        int newTokens = elapsed * bucket->rate;
        bucket->tokens = (bucket->tokens + newTokens > bucket->capacity) ? 
                        bucket->capacity : bucket->tokens + newTokens;
        bucket->lastUpdate = now;
    }
}

// 消费令牌
bool consumeTokens(TokenBucket* bucket, int count) {
    pthread_mutex_lock(&bucket->mutex);
    
    updateTokenBucket(bucket);
    
    if (bucket->tokens >= count) {
        bucket->tokens -= count;
        pthread_mutex_unlock(&bucket->mutex);
        return true;
    }
    
    pthread_mutex_unlock(&bucket->mutex);
    return false;
}
```

```c
// 测量网络延迟
double measureLatency(const char* host, int port) {
    struct timespec start, end;
    
    // 记录开始时间
    clock_gettime(CLOCK_MONOTONIC, &start);
    
    int sockfd = socket(AF_INET, SOCK_STREAM, 0);
    if (sockfd < 0) {
        return -1;
    }
    
    struct sockaddr_in addr;
    addr.sin_family = AF_INET;
    inet_pton(AF_INET, host, &addr.sin_addr);
    addr.sin_port = htons(port);
    
    int result = connect(sockfd, (struct sockaddr*)&addr, sizeof(addr));
    
    // 记录结束时间
    clock_gettime(CLOCK_MONOTONIC, &end);
    
    close(sockfd);
    
    if (result < 0) {
        return -1;
    }
    
    // 计算延迟（毫秒）
    double latency = (end.tv_sec - start.tv_sec) * 1000.0 + 
                    (end.tv_nsec - start.tv_nsec) / 1000000.0;
    
    return latency;
}

        if (latency > stats->max) stats->max = latency;
    }
    
    stats->sum += latency;
    stats->count++;
    stats->avg = stats->sum / stats->count;
}
 ```

```c
// 校验和计算
uint16_t calculateChecksum(uint8_t *data, int length) {
    uint32_t sum = 0;
    
    // 将数据按16位分组求和
    for (int i = 0; i < length - 1; i += 2) {
        sum += (data[i] << 8) + data[i + 1];
    }
    
    // 处理奇数长度
    if (length % 2 == 1) {
        sum += data[length - 1] << 8;
    }
    
    // 处理进位
    while (sum >> 16) {
        sum = (sum & 0xFFFF) + (sum >> 16);
    }
    
    return ~sum; // 取反
}
```

```c
// 滑动窗口协议
#define WINDOW_SIZE 4
#define MAX_SEQ 7

typedef struct {
    int seqNum;
    char data[1024];
    int dataLen;
    uint16_t checksum;
    bool acked;
    time_t sendTime;
} WindowFrame;

void goBackNSender(int sockfd, char *data, int totalLen) {
    WindowFrame window[WINDOW_SIZE];
    int base = 0;           // 窗口基序号
    int nextSeqNum = 0;     // 下一个发送序号
    int dataOffset = 0;     // 数据偏移量
    
    // 初始化窗口
    for (int i = 0; i < WINDOW_SIZE; i++) {
        window[i].acked = true;
    }
    
    while (dataOffset < totalLen || base != nextSeqNum) {
        // 发送窗口内未发送的帧
        while (nextSeqNum < base + WINDOW_SIZE && dataOffset < totalLen) {
            int windowIndex = nextSeqNum % WINDOW_SIZE;
            
            // 准备帧
            window[windowIndex].seqNum = nextSeqNum;
            int chunkSize = min(1024, totalLen - dataOffset);
            memcpy(window[windowIndex].data, data + dataOffset, chunkSize);
            window[windowIndex].dataLen = chunkSize;
            window[windowIndex].checksum = calculateChecksum((uint8_t*)&window[windowIndex], 
                                                           sizeof(WindowFrame) - sizeof(uint16_t) - sizeof(bool) - sizeof(time_t));
            window[windowIndex].acked = false;
            window[windowIndex].sendTime = time(NULL);
            
            // 发送帧
            send(sockfd, &window[windowIndex], sizeof(WindowFrame), 0);
            printf("发送帧 %d\n", nextSeqNum);
            
            nextSeqNum++;
            dataOffset += chunkSize;
        }
        
        // 检查ACK
        fd_set readfds;
        struct timeval timeout;
        FD_ZERO(&readfds);
        FD_SET(sockfd, &readfds);
        timeout.tv_sec = 0;
        timeout.tv_usec = 100000; // 100ms
        
        int activity = select(sockfd + 1, &readfds, NULL, NULL, &timeout);
        
        if (activity > 0 && FD_ISSET(sockfd, &readfds)) {
            Ack ack;
            if (recv(sockfd, &ack, sizeof(ack), 0) > 0) {
                printf("收到ACK %d\n", ack.ackNum);
                
                // 累积确认
                while (base <= ack.ackNum) {
                    window[base % WINDOW_SIZE].acked = true;
                    base++;
                }
            }
        }
        
        // 检查超时
        time_t currentTime = time(NULL);
        for (int i = base; i < nextSeqNum; i++) {
            int windowIndex = i % WINDOW_SIZE;
            if (!window[windowIndex].acked && 
                currentTime - window[windowIndex].sendTime > 2) {
                // 超时，回退重传
                printf("帧 %d 超时，回退重传\n", i);
                nextSeqNum = base;
                dataOffset -= (nextSeqNum - base) * 1024;
                if (dataOffset < 0) dataOffset = 0;
                break;
            }
        }
    }
}
```

```c
                break;
                
            case TRANSMITTING:
                // 边传输边检测冲突
                if (detectCollision()) {
                    printf("节点 %d: 检测到冲突\n", nodeId);
                    sendJamSignal(); // 发送干扰信号
                    state = COLLISION_DETECTED;
                } else {
                    // 传输完成
                    printf("节点 %d: 传输完成\n", nodeId);
                    state = IDLE;
                    dataLen = 0; // 数据发送完毕
                }
                break;
                
            case COLLISION_DETECTED:
                collisionCount++;
                if (collisionCount > 16) {
                    printf("节点 %d: 冲突次数过多，放弃传输\n", nodeId);
                    return;
                }
                
                // 计算退避时间（二进制指数退避）
                int k = min(collisionCount, 10);
                backoffTime = rand() % (1 << k);
                printf("节点 %d: 退避 %d 个时隙\n", nodeId, backoffTime);
                state = BACKOFF;
                break;
                
            case BACKOFF:
                if (backoffTime > 0) {
                    backoffTime--;
                    usleep(51200); // 一个时隙时间
                } else {
                    state = IDLE;
                }
                break;
        }
    }
}
```

```c
// 令牌环协议
typedef struct {
    uint8_t type;      // 0: 令牌, 1: 数据帧
    uint8_t priority;
    uint16_t destAddr;
    uint16_t srcAddr;
    char data[1024];

void tokenRingNode(int nodeId, int sockfd) {
    TokenRingFrame frame;
    bool hasToken = (nodeId == 0); // 节点0初始持有令牌
    queue<char*> sendQueue; // 待发送数据队列
    
    while (1) {
        if (hasToken) {
            // 持有令牌，检查是否有数据要发送
            if (!sendQueue.empty()) {
                // 发送数据
                char *data = sendQueue.front();
                sendQueue.pop();
                
                frame.type = 1; // 数据帧
                frame.priority = 0;
                frame.destAddr = getDestAddr(data);
                frame.srcAddr = nodeId;
                strcpy(frame.data, data);
                frame.dataLen = strlen(data);
                frame.frameStatus = 0;
                
                printf("节点 %d: 发送数据帧到节点 %d\n", nodeId, frame.destAddr);
                sendFrame(sockfd, &frame);
                
                free(data);
                hasToken = false;
            } else {
                // 没有数据发送，传递令牌
                frame.type = 0; // 令牌
                frame.priority = 0;
                
                printf("节点 %d: 传递令牌\n", nodeId);
                sendFrame(sockfd, &frame);
                hasToken = false;
            }
        } else {
            // 等待接收帧
            if (receiveFrame(sockfd, &frame)) {
                if (frame.type == 0) {
                    // 收到令牌
                    printf("节点 %d: 收到令牌\n", nodeId);
                    hasToken = true;
                } else {
                    // 收到数据帧
                    if (frame.destAddr == nodeId) {
                        // 目标是本节点
                        printf("节点 %d: 收到来自节点 %d 的数据: %s\n", 
                               nodeId, frame.srcAddr, frame.data);
                        frame.frameStatus |= 0x80; // 设置地址识别位
                        frame.frameStatus |= 0x40; // 设置帧复制位
                    }
                    
                    // 转发帧
                    sendFrame(sockfd, &frame);
                    
                    // 如果是自己发送的帧返回，则获得令牌
                    if (frame.srcAddr == nodeId) {
                        printf("节点 %d: 自己的帧返回，获得令牌\n", nodeId);
                        hasToken = true;
                    }
                }
            }
        }
    }
}
```

```c
uint16_t calculateIPChecksum(IPv4Header* header) {
    uint16_t* ptr = (uint16_t*)header;
    uint32_t sum = 0;
    int length = (header->version_ihl & 0x0F) * 4; // 首部长度
    
    // 校验和字段置零
    header->checksum = 0;
    
    // 计算校验和
    for (int i = 0; i < length / 2; i++) {
        sum += ntohs(ptr[i]);
    }
    
    // 处理进位
    while (sum >> 16) {
        sum = (sum & 0xFFFF) + (sum >> 16);
    }
    
    return htons(~sum);
}
```

### 5.3 路由算法

#### 5.3.1 距离向量算法（RIP）

```c
#define MAX_NODES 100
#define INFINITY 16

typedef struct {
    int dest;
    int distance;
    int nextHop;
} RouteEntry;

typedef struct {
    int nodeId;
    RouteEntry routes[MAX_NODES];
    int routeCount;
} RoutingTable;

// 距离向量算法
void distanceVectorRouting(RoutingTable* table, int nodeId) {
    bool updated = true;
    
    while (updated) {
        updated = false;
        
        // 从所有邻居接收距离向量
        for (int neighbor = 0; neighbor < MAX_NODES; neighbor++) {
            if (isNeighbor(nodeId, neighbor)) {
                RoutingTable neighborTable;
                receiveDistanceVector(neighbor, &neighborTable);
                
                // 更新路由表
                for (int i = 0; i < neighborTable.routeCount; i++) {
                    int dest = neighborTable.routes[i].dest;
                    int newDistance = getDirectDistance(nodeId, neighbor) + 
                                    neighborTable.routes[i].distance;
                    
                    // 查找现有路由
                    int routeIndex = findRoute(table, dest);
                    
                    if (routeIndex == -1) {
                        // 新路由
                        if (newDistance < INFINITY) {
                            addRoute(table, dest, newDistance, neighbor);
                            updated = true;
                        }
                    } else {
                        // 更新现有路由
                        if (newDistance < table->routes[routeIndex].distance) {
                            table->routes[routeIndex].distance = newDistance;
                            table->routes[routeIndex].nextHop = neighbor;
                            updated = true;
                        }
                    }
                }
            }
        }
        
        // 发送更新的距离向量给邻居
        if (updated) {
            for (int neighbor = 0; neighbor < MAX_NODES; neighbor++) {
                if (isNeighbor(nodeId, neighbor)) {
                    sendDistanceVector(neighbor, table);
                }
            }
        }
    }
}
```

#### 5.3.2 链路状态算法（OSPF）

```c
#define MAX_VERTICES 100

typedef struct {
    int vertex;
    int weight;
} Edge;

typedef struct {
    int vertexCount;
    int edgeCount;
    Edge edges[MAX_VERTICES][MAX_VERTICES];
    int adjacencyMatrix[MAX_VERTICES][MAX_VERTICES];
} Graph;

// Dijkstra算法实现
void dijkstra(Graph* graph, int source, int distances[], int previous[]) {
    bool visited[MAX_VERTICES] = {false};
    
    // 初始化距离
    for (int i = 0; i < graph->vertexCount; i++) {
        distances[i] = INFINITY;
        previous[i] = -1;
    }
    distances[source] = 0;
    
    for (int count = 0; count < graph->vertexCount - 1; count++) {
        // 找到未访问的最小距离顶点
        int minDistance = INFINITY;
        int minVertex = -1;
        
        for (int v = 0; v < graph->vertexCount; v++) {
            if (!visited[v] && distances[v] < minDistance) {
                minDistance = distances[v];
                minVertex = v;
            }
        }
        
        if (minVertex == -1) break;
        
        visited[minVertex] = true;
        
        // 更新邻接顶点的距离
        for (int v = 0; v < graph->vertexCount; v++) {
            if (!visited[v] && 
                graph->adjacencyMatrix[minVertex][v] != INFINITY &&
                distances[minVertex] + graph->adjacencyMatrix[minVertex][v] < distances[v]) {
                
                distances[v] = distances[minVertex] + graph->adjacencyMatrix[minVertex][v];
                previous[v] = minVertex;
            }
        }
    }
}

// 构建路由表
void buildRoutingTable(Graph* graph, int nodeId, RoutingTable* table) {
    int distances[MAX_VERTICES];
    int previous[MAX_VERTICES];
    
    dijkstra(graph, nodeId, distances, previous);
    
    table->nodeId = nodeId;
    table->routeCount = 0;
    
    for (int dest = 0; dest < graph->vertexCount; dest++) {
        if (dest != nodeId && distances[dest] != INFINITY) {
            // 找到下一跳
            int nextHop = dest;
            while (previous[nextHop] != nodeId && previous[nextHop] != -1) {
                nextHop = previous[nextHop];
            }
            
            table->routes[table->routeCount].dest = dest;
            table->routes[table->routeCount].distance = distances[dest];
            table->routes[table->routeCount].nextHop = nextHop;
            table->routeCount++;
        }
    }
}
```

```c
void ping(const char* targetIP, int count) {

    if (sockfd < 0) {
        perror("socket");
        return;
    }

    target.sin_family = AF_INET;

        packet.type = ICMP_ECHO_REQUEST;
        packet.code = 0;
        packet.checksum = 0;

            perror("sendto");
            continue;
        }
        
        // 接收回复
        char buffer[1024];
        struct sockaddr_in from;
        socklen_t fromLen = sizeof(from);
        
        struct timeval timeout;
        timeout.tv_sec = 5;
        timeout.tv_usec = 0;
        setsockopt(sockfd, SOL_SOCKET, SO_RCVTIMEO, &timeout, sizeof(timeout));
        
        ssize_t bytes = recvfrom(sockfd, buffer, sizeof(buffer), 0,
                                (struct sockaddr*)&from, &fromLen);

            struct timeval recvTime;
            gettimeofday(&recvTime, NULL);
            
            // 解析IP头部
            IPv4Header* ipHeader = (IPv4Header*)buffer;
            int ipHeaderLen = (ipHeader->version_ihl & 0x0F) * 4;
            
            // 解析ICMP头部
            ICMPPacket* icmpReply = (ICMPPacket*)(buffer + ipHeaderLen);
            
            if (icmpReply->type == ICMP_ECHO_REPLY && 
                icmpReply->identifier == getpid() &&
                icmpReply->sequence == i) {
                
                // 计算往返时间
                struct timeval* sentTime = (struct timeval*)icmpReply->data;
                double rtt = (recvTime.tv_sec - sentTime->tv_sec) * 1000.0 +
                           (recvTime.tv_usec - sentTime->tv_usec) / 1000.0;
                
                printf("来自 %s 的回复: 字节=%ld 时间=%.2fms TTL=%d\n",
```

### 5.5 ARP协议

```c
// ARP协议实现
typedef struct {
    uint16_t hardwareType;
    uint16_t protocolType;
    uint8_t hardwareLen;
    uint8_t protocolLen;
    uint16_t operation;
    uint8_t senderMAC[6];
    uint32_t senderIP;
    uint8_t targetMAC[6];
    uint32_t targetIP;
} ARPPacket;

typedef struct {
    uint32_t ip;
    uint8_t mac[6];
    time_t timestamp;
} ARPEntry;

#define ARP_TABLE_SIZE 256
ARPEntry arpTable[ARP_TABLE_SIZE];
int arpTableSize = 0;

// ARP表查找
int findARPEntry(uint32_t ip) {
    for (int i = 0; i < arpTableSize; i++) {
        if (arpTable[i].ip == ip) {
            return i;
        }
    }
    return -1;
}

// 添加ARP表项
void addARPEntry(uint32_t ip, uint8_t* mac) {
    int index = findARPEntry(ip);
    
    if (index == -1 && arpTableSize < ARP_TABLE_SIZE) {
        index = arpTableSize++;
    }
    
    if (index != -1) {
        arpTable[index].ip = ip;
        memcpy(arpTable[index].mac, mac, 6);
        arpTable[index].timestamp = time(NULL);
    }
}

// 发送ARP请求
void sendARPRequest(int sockfd, uint32_t targetIP, uint32_t sourceIP, uint8_t* sourceMAC) {
    ARPPacket packet;
    
    packet.hardwareType = htons(1);     // 以太网
    packet.protocolType = htons(0x0800); // IPv4
    packet.hardwareLen = 6;
    packet.protocolLen = 4;
    packet.operation = htons(1);         // ARP请求
    
    memcpy(packet.senderMAC, sourceMAC, 6);
    packet.senderIP = sourceIP;
    memset(packet.targetMAC, 0, 6);      // 未知MAC地址
    packet.targetIP = targetIP;
    
    // 构造以太网帧
    uint8_t frame[1500];
    uint8_t broadcastMAC[6] = {0xFF, 0xFF, 0xFF, 0xFF, 0xFF, 0xFF};
    
    // 以太网头部
    memcpy(frame, broadcastMAC, 6);      // 目标MAC（广播）
    memcpy(frame + 6, sourceMAC, 6);     // 源MAC
    *(uint16_t*)(frame + 12) = htons(0x0806); // ARP类型
    
    // ARP数据
    memcpy(frame + 14, &packet, sizeof(packet));
    
    // 发送帧
    struct sockaddr_ll addr;
    memset(&addr, 0, sizeof(addr));
    addr.sll_family = AF_PACKET;
    addr.sll_protocol = htons(ETH_P_ARP);
    addr.sll_ifindex = getInterfaceIndex("eth0");
    
    sendto(sockfd, frame, 14 + sizeof(packet), 0, 
           (struct sockaddr*)&addr, sizeof(addr));
    
    printf("发送ARP请求: 查询 %s 的MAC地址\n", ipIntToString(targetIP));
}

// 处理ARP回复
void handleARPReply(ARPPacket* packet) {
    printf("收到ARP回复: %s 的MAC地址是 %02x:%02x:%02x:%02x:%02x:%02x\n",
           ipIntToString(packet->senderIP),
           packet->senderMAC[0], packet->senderMAC[1], packet->senderMAC[2],
           packet->senderMAC[3], packet->senderMAC[4], packet->senderMAC[5]);
    
    // 更新ARP表
    addARPEntry(packet->senderIP, packet->senderMAC);
}

// ARP解析
bool resolveMAC(uint32_t ip, uint8_t* mac) {
    // 首先查找ARP表
    int index = findARPEntry(ip);
    if (index != -1) {
        // 检查表项是否过期（5分钟）
        if (time(NULL) - arpTable[index].timestamp < 300) {
            memcpy(mac, arpTable[index].mac, 6);
            return true;
        }
    }
    
    // 发送ARP请求
    int sockfd = socket(AF_PACKET, SOCK_RAW, htons(ETH_P_ARP));
    if (sockfd < 0) {
        return false;
    }
    
    uint8_t sourceMAC[6];
    uint32_t sourceIP;
    getLocalMACAndIP(sourceMAC, &sourceIP);
    
    sendARPRequest(sockfd, ip, sourceIP, sourceMAC);
    
    // 等待回复
    struct timeval timeout;
    timeout.tv_sec = 3;
    timeout.tv_usec = 0;
    setsockopt(sockfd, SOL_SOCKET, SO_RCVTIMEO, &timeout, sizeof(timeout));
    
    uint8_t buffer[1500];
    ssize_t bytes = recv(sockfd, buffer, sizeof(buffer), 0);
    
    if (bytes > 0) {
        ARPPacket* reply = (ARPPacket*)(buffer + 14); // 跳过以太网头部
        
        if (ntohs(reply->operation) == 2 && reply->senderIP == ip) {
            handleARPReply(reply);
            memcpy(mac, reply->senderMAC, 6);
            close(sockfd);
            return true;
        }
    }
    
    close(sockfd);
    return false;
}
```

## 6. 传输层

传输层为应用程序提供端到端的通信服务，主要协议包括TCP和UDP。

### 6.1 传输层的功能

1. **端到端通信**：在源主机和目标主机的应用程序之间建立通信
2. **多路复用/分解**：允许多个应用程序同时使用网络
3. **错误检测和恢复**：确保数据的可靠传输
4. **流量控制**：防止发送方发送过快
5. **拥塞控制**：防止网络过载

### 6.2 UDP协议

UDP（User Datagram Protocol）是无连接的、不可靠的传输协议。

#### 6.2.1 UDP特点

- **无连接**：发送数据前不需要建立连接
- **不可靠**：不保证数据包的到达和顺序
- **开销小**：头部只有8字节
- **支持广播和组播**

#### 6.2.2 UDP头部格式

```c
typedef struct {
    uint16_t srcPort;     // 源端口
    uint16_t destPort;    // 目标端口
    uint16_t length;      // UDP长度
    uint16_t checksum;    // 校验和
} UDPHeader;

// UDP校验和计算
uint16_t calculateUDPChecksum(uint32_t srcIP, uint32_t destIP, 
                             UDPHeader* udpHeader, uint8_t* data, int dataLen) {
    // 构造伪头部
    struct {
        uint32_t srcIP;
        uint32_t destIP;
        uint8_t zero;
        uint8_t protocol;
        uint16_t udpLength;
    } pseudoHeader;
    
    pseudoHeader.srcIP = srcIP;
    pseudoHeader.destIP = destIP;
    pseudoHeader.zero = 0;
    pseudoHeader.protocol = 17; // UDP协议号
    pseudoHeader.udpLength = udpHeader->length;
    
    // 计算校验和
    uint32_t sum = 0;
    
    // 伪头部
    uint16_t* ptr = (uint16_t*)&pseudoHeader;
    for (int i = 0; i < sizeof(pseudoHeader) / 2; i++) {
        sum += ntohs(ptr[i]);
    }
    
    // UDP头部（校验和字段置零）
    udpHeader->checksum = 0;
    ptr = (uint16_t*)udpHeader;
    for (int i = 0; i < sizeof(UDPHeader) / 2; i++) {
        sum += ntohs(ptr[i]);
    }
    
    // 数据部分
    ptr = (uint16_t*)data;
    for (int i = 0; i < dataLen / 2; i++) {
        sum += ntohs(ptr[i]);
    }
    
    // 处理奇数长度
    if (dataLen % 2 == 1) {
        sum += data[dataLen - 1] << 8;
    }
    
    // 处理进位
    while (sum >> 16) {
        sum = (sum & 0xFFFF) + (sum >> 16);
    }
    
    return htons(~sum);
}
```

```c
        }
    }
    
    close(sockfd);
```

### 6.3 TCP协议

TCP（Transmission Control Protocol）是面向连接的、可靠的传输协议。

#### 6.3.1 TCP特点

- **面向连接**：通信前需要建立连接
- **可靠传输**：保证数据的正确性和顺序
- **流量控制**：防止接收方缓冲区溢出
- **拥塞控制**：防止网络拥塞
- **全双工通信**：双方可同时发送和接收数据

#### 6.3.2 TCP头部格式

```c
typedef struct {
    uint16_t srcPort;        // 源端口
    uint16_t destPort;       // 目标端口
```

#### 6.3.3 TCP连接建立（三次握手）

```c
// TCP三次握手状态
typedef enum {
    CLOSED,
    LISTEN,
    SYN_SENT,
    SYN_RECEIVED,
    ESTABLISHED,
    FIN_WAIT_1,
    FIN_WAIT_2,
    CLOSE_WAIT,
    CLOSING,
    LAST_ACK,
    TIME_WAIT
} TCPState;

// TCP连接控制块
typedef struct {
    TCPState state;
    uint32_t sendNext;       // 下一个发送序号
    uint32_t sendUnack;      // 未确认的最小序号
    uint32_t recvNext;       // 下一个期望接收序号
    uint16_t sendWindow;     // 发送窗口大小
    uint16_t recvWindow;     // 接收窗口大小
    uint32_t mss;            // 最大段大小
    struct timeval rto;      // 重传超时时间
} TCPControlBlock;

// 三次握手客户端
void tcpConnect(int sockfd, struct sockaddr_in* serverAddr, TCPControlBlock* tcb) {
    // 第一次握手：发送SYN
    tcb->state = SYN_SENT;
    tcb->sendNext = rand(); // 初始序列号
    
    TCPHeader synPacket;
    synPacket.srcPort = htons(getLocalPort());
    synPacket.destPort = serverAddr->sin_port;
    synPacket.seqNum = htonl(tcb->sendNext);
    synPacket.ackNum = 0;
    synPacket.headerLen = 0x50; // 20字节头部
    synPacket.flags = TCP_SYN;
    synPacket.windowSize = htons(8192);
    synPacket.checksum = 0;
    synPacket.urgentPtr = 0;
    
    sendTCPPacket(sockfd, &synPacket, NULL, 0, serverAddr);
    printf("发送SYN，seq=%u\n", tcb->sendNext);
    
    // 等待SYN+ACK
    TCPHeader response;
    if (receiveTCPPacket(sockfd, &response, NULL, 0) > 0) {
        if ((response.flags & (TCP_SYN | TCP_ACK)) == (TCP_SYN | TCP_ACK) &&
            ntohl(response.ackNum) == tcb->sendNext + 1) {
            
            printf("收到SYN+ACK，seq=%u, ack=%u\n", 
                   ntohl(response.seqNum), ntohl(response.ackNum));
            
            // 第三次握手：发送ACK
            tcb->recvNext = ntohl(response.seqNum) + 1;
            tcb->sendNext++;
            
            TCPHeader ackPacket;
            ackPacket.srcPort = synPacket.srcPort;
            ackPacket.destPort = synPacket.destPort;
            ackPacket.seqNum = htonl(tcb->sendNext);
            ackPacket.ackNum = htonl(tcb->recvNext);
            ackPacket.headerLen = 0x50;
            ackPacket.flags = TCP_ACK;
            ackPacket.windowSize = htons(8192);
            ackPacket.checksum = 0;
            ackPacket.urgentPtr = 0;
            
            sendTCPPacket(sockfd, &ackPacket, NULL, 0, serverAddr);
            printf("发送ACK，seq=%u, ack=%u\n", tcb->sendNext, tcb->recvNext);
            
            tcb->state = ESTABLISHED;
            printf("连接建立成功\n");
        }
    }
}
```

#### 6.3.4 TCP连接释放（四次挥手）

```c
// 四次挥手主动关闭端
void tcpClose(int sockfd, struct sockaddr_in* peerAddr, TCPControlBlock* tcb) {
    // 第一次挥手：发送FIN
    tcb->state = FIN_WAIT_1;
    
    TCPHeader finPacket;
    finPacket.srcPort = htons(getLocalPort());
    finPacket.destPort = peerAddr->sin_port;
    finPacket.seqNum = htonl(tcb->sendNext);
    finPacket.ackNum = htonl(tcb->recvNext);
    finPacket.headerLen = 0x50;
    finPacket.flags = TCP_FIN | TCP_ACK;
    finPacket.windowSize = htons(tcb->recvWindow);
    finPacket.checksum = 0;
    finPacket.urgentPtr = 0;
    
    sendTCPPacket(sockfd, &finPacket, NULL, 0, peerAddr);
    printf("发送FIN，seq=%u\n", tcb->sendNext);
    tcb->sendNext++;
    
    // 等待ACK
    TCPHeader response;
    if (receiveTCPPacket(sockfd, &response, NULL, 0) > 0) {
        if ((response.flags & TCP_ACK) && 
            ntohl(response.ackNum) == tcb->sendNext) {
            
            printf("收到ACK，进入FIN_WAIT_2\n");
            tcb->state = FIN_WAIT_2;
            
            // 等待对方的FIN
            if (receiveTCPPacket(sockfd, &response, NULL, 0) > 0) {
                if (response.flags & TCP_FIN) {
                    printf("收到FIN，seq=%u\n", ntohl(response.seqNum));
                    tcb->recvNext = ntohl(response.seqNum) + 1;
                    
                    // 发送最后的ACK
                    TCPHeader lastAck;
                    lastAck.srcPort = finPacket.srcPort;
                    lastAck.destPort = finPacket.destPort;
                    lastAck.seqNum = htonl(tcb->sendNext);
                    lastAck.ackNum = htonl(tcb->recvNext);
                    lastAck.headerLen = 0x50;
                    lastAck.flags = TCP_ACK;
                    lastAck.windowSize = htons(tcb->recvWindow);
                    lastAck.checksum = 0;
                    lastAck.urgentPtr = 0;
                    
                    sendTCPPacket(sockfd, &lastAck, NULL, 0, peerAddr);
                    printf("发送最后ACK，进入TIME_WAIT\n");
                    
                    tcb->state = TIME_WAIT;
                    
                    // TIME_WAIT状态等待2MSL
                    sleep(60); // 简化为60秒
                    tcb->state = CLOSED;
                    printf("连接关闭\n");
                }
            }
        }
    }
}
```
