# 瓶子图片生成项目

本项目专注于AI生成各种瓶子图片的提示词收集和优化，提供系统化的设计风格指南和专业的提示词模板。

## 🎯 项目特色

- **全行业覆盖**: 涵盖饮料、化妆品、医药、家居、工艺品等主要行业
- **多风格支持**: 6种主要设计风格，满足不同品牌定位需求
- **专业指导**: 详细的材料工艺、设计元素、市场定位指南
- **实用模板**: 即用型英文提示词，支持直接AI生成
- **持续更新**: 跟踪行业趋势，不断完善和扩展内容

## 📋 提示词格式

每个提示词文件包含：
1. **角色定义** - AI扮演的专业角色
2. **技能描述** - 具备的专业技能
3. **具体要求** - 生成图片的详细要求
4. **英文提示词** - 完整的英文提示词
5. **中文翻译** - 对应的中文说明
6. **设计理念** - 风格的核心设计思想
7. **材料工艺** - 推荐的材料和制作工艺
8. **适用产品** - 适合的产品类型和应用场景

## 🛠 材料工艺体系

### 主要材料类型
- **玻璃材料**: 高端护肤品、香水（质感优良、环保可回收）
- **塑料材料**: 日常护肤、彩妆（轻便耐用、成本较低）
- **金属材料**: 奢华包装、限量版（质感高级、工艺复杂）
- **软包装**: 面膜、试用装（便携性强、成本低廉）

### 新兴环保材料
- **生物基塑料**: PLA、PHA、淀粉基材料
- **回收再生**: PCR塑料、海洋塑料回收
- **天然材料**: 竹纤维、软木、木质纤维

### 表面处理工艺
- **磨砂效果**: 高级质感，防指纹
- **亮面效果**: 光泽感强，视觉冲击
- **雕刻纹理**: 触感丰富，防滑功能
- **渐变效果**: 视觉层次，现代感强

## 🔧 技术支持

### 常见问题解答

**Q: 如何选择合适的风格？**
A: 根据目标消费群体、品牌定位和产品特性选择。建议参考[化妆品行业风格总览](./化妆品行业/风格总览.md)中的市场定位指南。

**Q: 提示词可以自定义修改吗？**
A: 完全可以。提示词模板是基础框架，您可以根据具体需求调整颜色、材料、尺寸等参数。

**Q: 如何确保生成的设计符合生产要求？**
A: 建议在设计阶段就考虑材料特性、工艺限制和成本控制，参考各风格文档中的"材料工艺"部分。

**Q: 不同AI工具的效果有差异吗？**
A: 是的。建议根据需求选择：创意设计用Midjourney，产品设计用DALL-E 3，批量生成用Stable Diffusion。

### 技术参数建议
- **图像分辨率**: 建议1024x1024或更高
- **宽高比**: 产品展示建议3:4或4:3
- **生成步数**: 20-50步（质量与速度平衡）
- **引导强度**: 7-12（创意与控制平衡）
