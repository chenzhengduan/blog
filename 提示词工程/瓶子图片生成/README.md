# 瓶子图片生成项目

本项目专注于AI生成各种瓶子图片的提示词收集和优化。

## 目录结构

### 按行业分类
- **饮料行业** - 各种饮料瓶设计
- **化妆品行业** - 护肤品、香水等瓶子设计
- **医药行业** - 药瓶、保健品瓶设计
- **家居用品** - 清洁用品、洗护用品瓶子
- **工艺品** - 装饰瓶、艺术瓶设计

### 按风格分类
- **现代简约风格**
- **复古经典风格**
- **奢华高端风格**
- **可爱卡通风格**
- **工业设计风格**
- **自然环保风格**

## 提示词格式

每个提示词文件包含：
1. **角色定义** - AI扮演的专业角色
2. **技能描述** - 具备的专业技能
3. **具体要求** - 生成图片的详细要求
4. **英文提示词** - 完整的英文提示词
5. **中文翻译** - 对应的中文说明
