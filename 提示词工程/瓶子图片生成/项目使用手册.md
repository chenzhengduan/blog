## AI工具配置指南

### 支持的AI工具

#### Midjourney
**版本要求**：V5.2或更高版本
**配置建议**：
- 模型：`--v 5.2` 或 `--v 6`
- 质量：`--q 2`（高质量）
- 风格化：`--s 250-750`（根据需求调整）
- 宽高比：`--ar 3:4`（瓶子标准比例）

**优化参数**：
```
--chaos 10-30        # 增加创意变化
--weird 250-500      # 增加独特性
--stylize 250        # 平衡风格化程度
--quality 2          # 最高质量
```

#### DALL-E 3
**平台**：ChatGPT Plus、API
**配置建议**：
- 分辨率：1024x1024（标准）或1024x1792（竖版）
- 质量：HD
- 风格：Natural（自然）或Vivid（鲜艳）

**提示词长度**：建议控制在400字符以内

#### Stable Diffusion
**模型推荐**：
- SDXL 1.0（基础模型）
- Realistic Vision（写实风格）
- DreamShaper（创意风格）

**参数配置**：
```
Steps: 20-30
CFG Scale: 7-12
Sampler: DPM++ 2M Karras
Size: 768x1024 或 512x768
Seed: -1（随机）或固定值（复现）
```

#### Leonardo AI
**模型选择**：
- Leonardo Diffusion XL（通用）
- PhotoReal（写实）
- Anime Pastel Dream（动漫风格）

**配置参数**：
- Guidance Scale: 7-10
- Steps: 15-25
- Dimensions: 768x1024

### 工具选择建议

#### 按用途选择
- **商业用途**：Midjourney（版权友好）
- **快速原型**：DALL-E 3（速度快）
- **精细控制**：Stable Diffusion（参数丰富）
- **批量生成**：Leonardo AI（成本效益）

#### 按风格选择
- **写实风格**：DALL-E 3、Stable Diffusion
- **艺术风格**：Midjourney、Leonardo AI
- **卡通风格**：DALL-E 3、特定SD模型
- **概念设计**：Midjourney、Stable Diffusion

## 提示词优化技巧

### 基础提示词结构

#### 标准模板
```
[主体描述] + [风格定义] + [材质纹理] + [颜色方案] + [光照环境] + [构图角度] + [质量参数]
```

#### 示例分析
```
原始提示词：
"A modern minimalist perfume bottle"

优化后提示词：
"A sleek modern minimalist perfume bottle, frosted glass material with clean geometric lines, soft gradient from pearl white to light gray, studio lighting with soft shadows, centered composition on white background, professional product photography, ultra-high resolution, 8K quality"
```

### 关键词优化策略

#### 主体描述优化
**基础词汇**：
- bottle → elegant bottle, sophisticated container, premium vessel
- perfume bottle → luxury fragrance bottle, designer scent container
- cosmetic bottle → beauty product container, skincare vessel

**形状描述**：
- round → spherical, circular, orb-like
- square → cubic, geometric, angular
- tall → elongated, slender, statuesque
- curved → flowing, organic, sinuous

#### 材质描述增强
**玻璃类型**：
- clear glass → crystal-clear borosilicate glass
- frosted glass → etched matte glass with soft texture
- colored glass → tinted amber glass with warm undertones

**表面处理**：
- smooth → polished to mirror finish
- textured → subtly embossed with tactile patterns
- matte → velvety matte coating

#### 颜色描述精确化
**基础颜色**：
- blue → deep sapphire blue, ocean teal, powder blue
- red → crimson red, burgundy wine, coral pink
- gold → champagne gold, rose gold, antique brass

**颜色组合**：
- gradient → smooth gradient transition
- ombre → subtle ombre effect
- metallic → iridescent metallic sheen

### 风格特定优化

#### 现代简约风格
**核心关键词**：
```
minimalist, clean lines, geometric, sleek, contemporary, 
reduced form, functional beauty, understated elegance
```

**避免使用**：
```
ornate, decorative, baroque, vintage, cluttered
```

```
luxurious, premium, sophisticated, opulent, refined,
exquisite craftsmanship, precious materials, exclusive design
```

**材质强调**：
```
crystal glass, gold accents, platinum details,

```

#### 可爱卡通风格

```
playful, cute, whimsical, colorful, friendly,
rounded shapes, soft edges, cheerful, adorable
```

**色彩特点**：
```
pastel colors, bright hues, rainbow gradient,
candy colors, soft pink, sky blue
```

### 技术参数优化

#### 质量提升关键词
```
ultra-high resolution, 8K quality, professional photography,
```

## 质量控制标准

### 图片质量评估

#### 技术质量标准

**分辨率要求**：
- 最低：1024x1024像素
- 推荐：1536x1536像素或更高
- 商用：2048x2048像素以上

**清晰度评估**：
- 边缘锐利度：瓶子轮廓清晰可辨
- 细节保留：标签文字、纹理细节可见
- 焦点准确：主体对焦准确，背景适当虚化

**色彩质量**：
- 色彩准确：符合预期色彩方案
- 饱和度适中：不过饱和或过淡
- 色彩平衡：白平衡准确，无明显色偏

#### 设计质量标准

**造型评估**：
- 比例协调：符合人体工学和美学比例
- 结构合理：瓶身、瓶颈、瓶盖比例恰当
- 细节完整：所有设计元素完整呈现

**风格一致性**：
- 风格纯正：符合指定设计风格特征
- 元素统一：所有设计元素风格统一
- 品牌适配：符合目标品牌定位

**创新性评估**：
- 独特性：具有独特的设计特色
- 创意性：体现创新思维和创意
- 实用性：兼顾美观性和实用性

### 批量生成质量控制

#### 生成策略

**种子管理**：
- 记录优质种子：保存高质量图片的种子值
- 种子变化：在优质种子基础上微调
- 批次测试：小批量测试后大批量生成

**参数优化**：
- A/B测试：对比不同参数设置效果
- 渐进优化：逐步调整参数达到最佳效果
- 记录最佳：记录最佳参数组合

#### 筛选标准

**一级筛选（技术质量）**：
- 分辨率达标
- 无明显技术缺陷
- 主体完整清晰

**二级筛选（设计质量）**：
- 风格符合要求
- 造型比例协调
- 色彩搭配合理

**三级筛选（商业价值）**：
- 市场适应性
- 品牌契合度
- 创新独特性

### 后期处理标准

#### 基础调整

**色彩校正**：
- 白平衡调整
- 曝光补偿
- 对比度优化
- 饱和度微调

**清晰度增强**：
- 锐化处理（适度）
- 噪点降低
- 细节增强

#### 专业处理

**背景处理**：
- 背景替换或优化
- 阴影调整
- 反射效果添加

**产品精修**：
- 瑕疵修复
- 高光调整
- 材质质感增强

### 常见技术问题

#### AI工具相关

**Midjourney问题**：
- 问题：图片生成失败
- 解决：检查提示词长度，简化复杂描述
- 问题：风格不稳定
- 解决：使用固定种子值，调整stylize参数

**DALL-E问题**：
- 问题：内容政策限制
- 解决：避免敏感词汇，使用替代描述
- 问题：细节不够精细
- 解决：分步骤生成，后期合成处理

**Stable Diffusion问题**：
- 问题：显存不足
- 解决：降低分辨率，使用优化版本
- 问题：生成速度慢
- 解决：调整采样步数，选择快速采样器

### 术语表

**AI相关术语**：

- **Composition（构图）**：画面元素的排列组织
