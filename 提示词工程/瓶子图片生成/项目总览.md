# 瓶子图片生成项目 - 总览

## 项目简介

本项目是一个专业的AI图片生成提示词库，专注于各种瓶子设计的图片生成。项目按照行业分类和设计风格进行组织，每个提示词文件都包含完整的角色定义、技能描述、具体要求、英文提示词和中文翻译。

## 提示词文件结构

每个提示词文件都包含以下标准化内容：

### 1. 角色定义 (Role Definition)
- **中文描述**：AI扮演的专业角色
- **英文描述**：对应的英文角色定义

### 2. 技能描述 (Skills)
- **中文技能**：专业技能和能力描述
- **英文技能**：对应的英文技能说明

### 3. 具体要求 (Specific Requirements)
- **中文要求**：图片生成的具体要求
- **英文要求**：对应的英文要求说明

### 4. 英文提示词 (English Prompt)
- 完整的英文提示词，可直接用于AI图片生成
- 包含详细的设计元素、技术要求、风格指导

### 5. 中文翻译 (Chinese Translation)
- 英文提示词的完整中文翻译
- 便于理解和本地化使用

### 6. 扩展内容
- **参考资料**：相关的设计参考和案例
- **适用产品**：具体的产品类型和使用场景
- **使用建议**：实际应用的建议和注意事项

## 使用指南

### 快速开始
1. 根据你的产品类型选择对应的行业目录
2. 根据设计需求选择合适的风格文件
3. 复制英文提示词到AI图片生成工具
4. 根据需要调整具体参数和要求

### 自定义建议
- **品牌适配**：根据品牌色彩和调性调整色彩要求
- **产品特性**：根据产品特点调整材质和造型要求
- **市场定位**：根据目标用户群体调整风格倾向
- **使用场景**：根据使用环境调整背景和氛围

### 最佳实践
1. **组合使用**：可以结合多个风格的元素创造独特效果
2. **迭代优化**：根据生成结果逐步调整提示词参数
3. **A/B测试**：尝试不同的提示词变体对比效果
4. **保存记录**：记录成功的提示词组合便于复用

## 技术要求

### 推荐的AI工具
- **Midjourney**：适合艺术性和创意性较强的设计
- **DALL-E 3**：适合精确控制和商业用途
- **Stable Diffusion**：适合本地部署和自定义训练
- **Adobe Firefly**：适合商业用途和版权安全

### 输出规格建议
- **分辨率**：建议4K或更高分辨率
- **格式**：PNG（透明背景）或JPG（实景背景）
- **比例**：根据使用场景选择合适的宽高比
- **质量**：商业级别的图片质量
