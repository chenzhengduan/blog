# 提示词工程指南

## 概述

提示词工程是瓶子图片生成-Lovart项目的核心技术之一。高质量的提示词直接决定了AI工具生成结果的准确性和专业度。本指南详细阐述了提示词构建的原则、方法、模板和优化技巧。

## 提示词工程原则

### 1. 精确性原则
- **具体描述**：使用具体、明确的词汇，避免模糊表达
- **量化指标**：尽可能使用可量化的描述
- **专业术语**：使用行业标准术语和专业词汇
- **细节完整**：包含所有必要的设计细节

### 2. 层次性原则
- **重要性排序**：按重要性从高到低排列指令
- **逻辑结构**：遵循清晰的逻辑结构
- **分类组织**：将相关指令分组组织
- **优先级明确**：明确各部分的优先级

### 3. 一致性原则
- **风格统一**：保持整体风格的一致性
- **术语统一**：使用统一的专业术语
- **格式统一**：遵循统一的格式规范
- **逻辑一致**：确保各部分逻辑协调

### 4. 完整性原则
- **要素齐全**：包含所有必要的设计要素
- **约束明确**：明确所有设计约束和要求
- **目标清晰**：清楚表达设计目标和期望
- **上下文完整**：提供充分的上下文信息

## 提示词结构框架

### 标准结构模板

```
[1. 结构保持指令]
[2. 主要修改指令]
[3. 品牌元素指令]
[4. 环境设置指令]
[5. 风格定义指令]
[6. 质量要求指令]
```

### 详细结构说明

#### 1. 结构保持指令（Structure Preservation）
**目的**：确保原始产品的核心特征得到保留

**标准模板**：
```
"Keep the exact same [specific_elements] without any changes to [detailed_aspects]."
```

**常用表达**：
- `Keep the exact same bottle shape, proportions, and structure`
- `Maintain the original pump head design and mechanism`
- `Preserve all physical dimensions and proportions`
- `Do not alter the bottle's fundamental geometry`

**行业专用模板**：
- **化妆品**：`Keep the elegant bottle silhouette and luxury proportions`
- **医药**：`Maintain the standard pharmaceutical bottle structure and safety features`
- **饮料**：`Preserve the ergonomic bottle shape and grip design`

#### 2. 主要修改指令（Primary Modifications）
**目的**：明确指定需要修改的核心元素

**颜色修改模板**：
```
"Change the color scheme to [primary_colors] with [secondary_colors] accents, creating a [emotional_effect] feeling."
```

**材质修改模板**：
```
"Transform the surface to [material_type] with [texture_description] finish, enhancing [desired_quality]."
```

**常用颜色方案**：
- **清凉感**：`cool blue and silver tones`, `icy blue with white accents`
- **温暖感**：`warm amber and gold tones`, `rich brown with copper accents`
- **奢华感**：`deep black with gold details`, `pearl white with silver trim`
- **自然感**：`earth green with natural wood tones`, `sage green with cream accents`

#### 3. 品牌元素指令（Brand Integration）
**目的**：无缝整合品牌信息和产品特点

**品牌名称模板**：
```
"Create a clean, modern label design featuring the '[brand_name]' prominently in a [font_style] font suitable for [target_audience]."
```

**产品信息模板**：
```
"Include secondary text elements like '[key_features]' and '[product_benefits]' in a complementary typography style."
```

**字体风格选择**：
- **现代简约**：`clean sans-serif`, `minimalist geometric`
- **奢华高端**：`elegant serif`, `sophisticated script`
- **运动活力**：`bold condensed`, `dynamic angular`
- **传统经典**：`classic serif`, `traditional roman`

#### 4. 环境设置指令（Environment Design）
**目的**：创造合适的产品展示环境

**表面材质模板**：
```
"Place the bottle on a [surface_material] surface with [surface_characteristics]."
```

**环境元素模板**：
```
"Add [environmental_elements] around the product to emphasize [desired_effect] and enhance [brand_message]."
```

**光线设置模板**：
```
"Use [lighting_type] lighting with [color_temperature] undertones to create [mood_description]."
```

**常用环境配置**：
- **奢华感**：`polished marble surface`, `soft golden lighting`, `subtle reflections`
- **清凉感**：`frosted glass surface`, `cool blue lighting`, `water droplets`
- **自然感**：`wooden surface`, `warm natural lighting`, `organic elements`
- **专业感**：`clean white surface`, `neutral lighting`, `minimal distractions`

#### 5. 风格定义指令（Style Definition）
**目的**：确定整体设计风格和美学方向

**风格模板**：
```
"The overall style should be [primary_style], [secondary_style], and [tertiary_style], suitable for [target_market] and [usage_context]."
```

**风格关键词库**：
- **现代风格**：`minimalist`, `contemporary`, `sleek`, `geometric`
- **奢华风格**：`premium`, `sophisticated`, `elegant`, `luxurious`
- **自然风格**：`organic`, `eco-friendly`, `sustainable`, `natural`
- **运动风格**：`dynamic`, `energetic`, `bold`, `athletic`

#### 6. 质量要求指令（Quality Requirements）
**目的**：确保输出质量符合专业标准

**质量模板**：
```
"Ensure [quality_aspects] with [technical_requirements] and [professional_standards]."
```

**常用质量要求**：
- `photorealistic quality with sharp details`
- `professional product photography lighting`
- `high-resolution commercial-grade output`
- `color accuracy suitable for print production`

## 高级提示词技巧

### 1. 情感驱动技巧

#### 情感词汇库
- **信任感**：`trustworthy`, `reliable`, `professional`, `clinical`
- **奢华感**：`luxurious`, `premium`, `sophisticated`, `exclusive`
- **清新感**：`fresh`, `clean`, `pure`, `invigorating`
- **活力感**：`energetic`, `dynamic`, `vibrant`, `powerful`

#### 情感组合策略
```
"Creating a [primary_emotion] feeling that evokes [secondary_emotion] and appeals to [target_psychology]."
```

### 2. 对比强化技巧

#### 对比模板
```
"Transform from [current_state] to [desired_state], emphasizing the contrast between [old_characteristics] and [new_characteristics]."
```

#### 常用对比
- **温度对比**：`from warm tones to cool refreshing colors`
- **质感对比**：`from matte finish to glossy premium surface`
- **风格对比**：`from casual appearance to professional presentation`

### 3. 细节放大技巧

#### 细节强调模板
```
"Pay special attention to [specific_detail], ensuring [quality_requirement] and [visual_impact]."
```

#### 关键细节
- **标签细节**：`crisp text rendering`, `perfect alignment`, `color accuracy`
- **材质细节**：`realistic texture`, `appropriate reflectivity`, `surface authenticity`
- **光影细节**：`natural shadows`, `realistic highlights`, `depth perception`

### 4. 约束优化技巧

#### 强约束表达
```
"MUST maintain [critical_elements]" / "NEVER change [protected_elements]" / "ENSURE [quality_standards]"
```

#### 软约束表达
```
"Preferably keep [preferred_elements]" / "Ideally enhance [desired_aspects]" / "Consider [optional_improvements]"
```

## 质量控制与优化

### 提示词质量评估

#### 评估维度
1. **完整性**：是否包含所有必要元素
2. **准确性**：描述是否准确无歧义
3. **一致性**：各部分是否协调一致
4. **可执行性**：AI工具是否能准确理解

#### 评估清单
- [ ] 结构保持指令明确
- [ ] 修改要求具体详细
- [ ] 品牌元素完整准确
- [ ] 环境设置合理
- [ ] 风格定义清晰
- [ ] 质量要求明确
- [ ] 逻辑结构清晰
- [ ] 语言表达准确

### 迭代优化策略

#### 问题诊断
1. **结果偏差分析**：对比期望与实际结果
2. **关键词效果评估**：分析关键词的实际效果
3. **结构逻辑检查**：检查提示词结构是否合理
4. **语言表达优化**：改进模糊或不准确的表达

#### 优化方法
1. **增强关键词**：加强重要指令的表达
2. **细化描述**：增加关键细节的描述
3. **调整顺序**：优化指令的排列顺序
4. **简化表达**：去除冗余和模糊表达

### 成功案例分析

#### 案例1：男士洗发水重设计
**原始需求**：重新设计洗发水瓶子，品牌"穆宸"，男士夏季清凉去屑

**优化前提示词**：
```
Change the bottle to blue color for men's shampoo with brand name 穆宸.
```

- 整体质量：专业级别

## 常见问题与解决方案

### Q1: 提示词太长，如何优化？
**解决方案**：
1. 使用标准化模板减少重复
2. 合并相似指令
3. 去除冗余描述
4. 使用更精确的词汇

### Q2: AI理解偏差，如何改进？
**解决方案**：
1. 使用更具体的描述
2. 避免歧义表达
3. 增加约束条件
4. 使用标准术语

### Q3: 结果不稳定，如何提高一致性？
**解决方案**：
1. 标准化提示词模板
2. 固定关键参数
3. 建立质量检查机制
4. 积累成功案例

### Q4: 如何处理复杂需求？
**解决方案**：
1. 分解复杂需求
2. 分层构建提示词
3. 使用优先级排序
4. 多轮迭代优化
