# 工具配置指南

## 概述

瓶子图片生成-Lovart项目使用两个核心AI工具来完成专业的产品包装设计任务。本文档详细说明了每个工具的功能特点、使用场景、参数配置和最佳实践。

## 核心工具

### 1. Image Analyzer（图像分析工具）

#### 工具概述
Image Analyzer是专业的图像分析工具，能够深度分析图像内容，提取关键特征和设计元素，为后续的设计工作提供准确的基础信息。

#### 主要功能
- **结构分析**：识别瓶子的形状、比例、结构特征
- **材质识别**：分析表面材质、质感、透明度等
- **色彩分析**：提取主要色彩方案和色彩搭配
- **品牌元素识别**：识别现有标签、文字、图案等
- **设计风格分析**：判断整体设计风格和美学特征
- **技术参数提取**：分析图像质量、拍摄角度、光线条件

#### 使用场景
- 项目开始前的原始图像分析
- 需要深入了解产品特征时
- 制定设计策略前的信息收集
- 质量控制阶段的结果对比

#### 调用方法
```
工具名称：image_analyzer
参数：
- image_url: 图像URL地址
- analysis_focus: 分析重点（可选）
```

#### 最佳实践
1. **高质量输入**：确保原始图像清晰、光线充足
2. **明确分析目标**：根据项目需求指定分析重点
3. **结果记录**：详细记录分析结果，为后续设计提供参考
4. **多角度分析**：必要时从不同角度分析同一产品

### 2. Flux Kontext Max（图像生成工具）

#### 工具概述
Flux Kontext Max是高质量的图像生成工具，特别擅长基于现有图像进行精确修改和优化，能够在保持原始结构的同时实现专业的设计改进。

#### 主要功能
- **结构保持**：精确保持原始产品的形状和比例
- **精细修改**：对颜色、材质、标签等进行精确调整
- **环境重构**：重新设计产品展示环境和背景
- **品牌整合**：无缝整合新的品牌元素和信息
- **风格转换**：在保持产品特征的同时转换设计风格
- **质量优化**：提升图像整体质量和专业度

#### 使用场景
- 产品包装重新设计
- 品牌形象升级
- 市场定位调整
- 季节性设计变更
- 功效突出设计

#### 参数配置

##### 基础参数
```
工具名称：flux_kontext_max
必需参数：
- english_prompt: 详细的英文提示词
- image_name: 图像名称（建议使用中文）
- image_url_list: 原始图像URL列表

可选参数：
- aspect_ratio: 图像比例（默认："9:16"）
- rescale: 是否调整大小（默认：false）
```

##### 提示词构建规范

**1. 结构保持指令**
```
"Keep the exact same bottle shape, proportions, pump head design, and structure without any changes to the physical details."
```

**2. 修改指令模板**
```
"Change the [element] to [new_specification] while maintaining [preserved_aspects]."

示例：
"Change the color scheme to cool blue and silver tones while maintaining the original bottle shape and pump design."
```

**3. 品牌整合指令**
```
"Create a clean, modern label design featuring the '[brand_name]' prominently in a [style] font. Include secondary text elements like '[key_features]'."

示例：
"Create a clean, modern label design featuring the '穆宸' brand name prominently in a bold, masculine font. Include secondary text elements like '冰爽清凉' and '去屑止痒'."
```

**4. 环境设置指令**
```
"Place the bottle on a [surface_type] with [environmental_elements] to emphasize [desired_effect]. Use [lighting_type] lighting with [color_tone] undertones."

示例：
"Place the bottle on a light blue marble surface with water droplets around the base to emphasize the cooling effect. Use soft, natural lighting with blue undertones."
```

**5. 风格定义指令**
```
"The overall style should be [style_keywords], suitable for [target_audience] and [product_category]."

示例：
"The overall style should be minimalist, modern, masculine, and premium, suitable for high-end men's grooming products."
```

## 工具协作流程

### 标准协作流程
1. **Image Analyzer分析** → 获取产品特征信息
2. **需求分析** → 确定设计目标和约束
3. **策略制定** → 基于分析结果制定设计策略
4. **Flux Kontext Max生成** → 执行设计生成
5. **结果评估** → 评估生成结果质量
6. **优化迭代** → 根据评估结果优化改进

### 高效协作技巧
1. **信息传递**：确保分析结果准确传递到生成阶段
2. **参数联动**：根据分析结果优化生成参数
3. **质量闭环**：建立分析-生成-评估的质量闭环
4. **经验积累**：记录成功案例和优化经验

## 常见问题与解决方案

### Image Analyzer常见问题

**Q: 分析结果不够详细怎么办？**
A: 可以指定analysis_focus参数，明确分析重点，或者提供更高质量的原始图像。

**Q: 如何提高分析准确性？**
A: 确保图像清晰、光线充足、角度合适，避免遮挡和模糊。

### Flux Kontext Max常见问题

**Q: 生成结果与原图差异太大怎么办？**
A: 加强结构保持指令，明确指定需要保留的具体特征。

**Q: 品牌元素不够清晰怎么办？**
A: 在提示词中强调品牌元素的重要性，使用"prominently"、"clearly visible"等强调词。

**Q: 色彩效果不理想怎么办？**
A: 详细描述期望的色彩方案，包括主色调、辅助色、色彩情感等。

**Q: 如何提高生成速度？**
A: 优化提示词长度，避免冗余描述，使用标准化的指令模板。
