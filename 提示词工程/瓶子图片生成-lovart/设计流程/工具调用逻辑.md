# 工具调用逻辑

## 概述

本文档定义了瓶子图片生成-Lovart项目中各种工具的调用逻辑、参数配置和使用策略，确保工具使用的标准化和效果最大化。

## 工具体系架构

### 核心工具分类
```
主要生成工具:
├── Flux Kontext Max (主力图像生成工具)
├── Poster Generator (海报生成工具)
└── HTML Generate (HTML布局工具)

分析工具:
├── Image Analyzer (图像分析工具)
└── Poster Design Directions (设计方向生成)

辅助工具:
├── Terminate (任务结束工具)
├── Terminate with Questions (问题收集工具)
└── Handoff (任务移交工具)
```

### 工具选择决策树
```
任务类型判断:
├── 基于现有图像的产品设计
│   ├── 保持产品结构 → Flux Kontext Max
│   └── 完全重新设计 → Flux Kontext Max + 创新提示词
├── 海报设计任务
│   ├── 文字内容较多 → HTML Generate
│   └── 图像为主 → Poster Generator
└── 纯分析任务
    └── 图像特征分析 → Image Analyzer
```

## Flux Kontext Max 工具

### 适用场景
```
最佳适用场景:
1. 基于现有瓶子图像的重新设计
2. 保持产品形状和结构的前提下修改外观
3. 品牌形象升级和视觉优化
4. 材质、色彩、标签的专业化调整
5. 产品展示环境的优化设置

不适用场景:
1. 完全创新的产品造型设计
2. 复杂的多产品组合展示
3. 需要精确文字排版的设计
4. 动态效果或动画需求
```

### 参数配置策略

#### 基础参数设置
```
必需参数:
- english_prompt: 详细的英文提示词 (核心参数)
- image_name: 图像名称 (使用中文，与用户语言一致)
- image_url_list: 原始图像URL列表 (基础图像)

重要参数:
- aspect_ratio: 图像比例
  * 9:16 (推荐) - 产品展示的标准比例
  * 16:9 - 横向展示或环境场景
  * 1:1 - 社交媒体或方形展示
  * 3:4 - 传统产品摄影比例

- rescale: 图像缩放
  * false (推荐) - 保持原始比例和细节
  * true - 需要调整尺寸时使用
```

#### 高级参数优化
```
质量控制参数:
- quality: 图像质量等级
  * high (推荐) - 专业设计需求
  * medium - 快速预览需求
  * ultra - 最高质量要求

- style_strength: 风格强度
  * 0.7-0.9 (推荐) - 平衡原图特征和新风格
  * 0.5-0.7 - 保守修改，更多保留原图
  * 0.9-1.0 - 激进修改，更多新风格

- seed: 随机种子
  * 固定值 - 确保结果可重现
  * 随机值 - 获得多样化结果
```

### 提示词构建策略

#### 标准提示词结构
```
1. 角色定义部分 (20-30词)
   "You are a professional product packaging designer specializing in [产品类型] bottles..."

2. 保留指令部分 (30-50词)
   "Keep the exact same bottle shape, proportions, [具体结构特征] without any changes to the physical details..."

3. 修改指令部分 (50-80词)
   "Change the [具体元素] to [目标效果]. The [材质/色彩/质感] should be [详细描述]..."

4. 新增元素部分 (40-60词)
   "Add [新元素] featuring [具体要求]. Include [品牌信息/文字内容] in [字体风格] font..."

5. 环境设置部分 (30-50词)
   "Place the bottle on [表面材质] with [环境元素]. Use [光线类型] lighting with [氛围描述]..."

6. 风格定义部分 (20-40词)
   "The overall style should be [风格描述], suitable for [目标受众] and [使用场景]..."
```

#### 提示词优化技巧
```
精确性技巧:
1. 使用具体的材质名称: "frosted glass" vs "textured surface"
2. 明确色彩描述: "cool blue (#4A90E2)" vs "blue color"
3. 详细的质感描述: "matte metallic silver finish" vs "silver color"
4. 精确的位置描述: "centered on the upper third" vs "on top"

强调技巧:
1. 重要元素大写: "ENTIRE logo", "CLEARLY visible"
2. 重复关键词: "modern, sleek, contemporary design"
3. 使用强调词: "ensure", "make sure", "must be"
4. 否定描述: "without any changes", "do not modify"

专业术语:
1. 材质术语: "brushed aluminum", "injection-molded plastic"
2. 工艺术语: "UV coating", "embossed texture"
3. 设计术语: "minimalist aesthetic", "ergonomic design"
4. 摄影术语: "soft diffused lighting", "shallow depth of field"
```

### 常见问题解决

#### 结果不符合预期的处理
```
问题类型及解决方案:

1. 产品形状变形
   解决: 加强保留指令
   "Maintain the EXACT original bottle silhouette and proportions"

2. 标识不清晰
   解决: 强调标识可见性
   "Make sure the ENTIRE brand logo is clearly visible and legible"

3. 色彩不准确
   解决: 使用色彩代码
   "Use specific color codes: primary blue (#1E3A8A), accent silver (#C0C0C0)"

4. 材质质感不真实
   解决: 详细材质描述
   "Realistic frosted glass texture with subtle light diffusion"

5. 整体风格不统一
   解决: 强化风格描述
   "Consistently modern minimalist style throughout all elements"
```

#### 迭代优化策略
```
优化原则:
1. 单次修改1-2个主要问题
2. 保持成功元素不变
3. 逐步细化和完善
4. 记录每次修改内容

迭代控制:
- 最大迭代次数: 3次
- 每次迭代间隔: 充分分析上次结果
- 停止条件: 达到满意效果或无明显改进
```

## Image Analyzer 工具

### 使用时机
```
必须使用的情况:
1. 接收到用户提供的原始图像
2. 需要详细了解产品特征
3. 制定设计策略前的分析
4. 验证生成结果的准确性

可选使用的情况:
1. 对比分析不同设计方案
2. 深度分析特定设计元素
3. 质量评估和改进建议
```

### 分析重点配置
```
标准分析重点:
1. 产品形态和结构特征
2. 材质和表面处理工艺
3. 色彩方案和搭配效果
4. 品牌元素和文字信息
5. 整体设计风格和美学

深度分析重点:
1. 细节工艺和制造特征
2. 功能性设计和人体工学
3. 文化内涵和象征意义
4. 市场定位和竞争分析
```

### 结果应用策略
```
分析结果的使用:
1. 设计约束识别: 确定不可改变的元素
2. 优化机会发现: 识别可以改进的方面
3. 风格定位参考: 确定设计方向和风格
4. 技术实现指导: 为提示词提供准确信息
```

## Terminate 系列工具

### Terminate 工具
```
使用场景:
1. 设计任务完成，结果满足要求
2. 提供了完整的设计方案
3. 用户表示满意或无需进一步修改

使用要求:
- 提供简洁的任务总结
- 强调设计的核心价值
- 给出使用建议或注意事项
```

### Terminate with Questions 工具
```
使用场景:
1. 用户需求信息不完整
2. 需要澄清设计要求
3. 存在多种设计方向选择

问题设计原则:
- 问题数量: 3-5个关键问题
- 问题类型: 选择题优于开放题
- 语言要求: 简单明了，避免专业术语
- 逻辑顺序: 从基础到具体，从重要到次要

标准问题模板:
1. 产品定位: "这个产品主要面向哪个群体？(男性/女性/儿童/全年龄)"
2. 风格偏好: "您希望什么样的设计风格？(现代简约/奢华高端/自然环保)"
3. 色彩偏好: "您偏好什么色调？(冷色调/暖色调/中性色调)"
4. 功能重点: "最希望突出产品的哪个特点？(功效/品质/价格/创新)"
5. 使用场景: "主要在什么场合使用？(日常/专业/礼品/收藏)"
```

## 工具组合使用策略

### 标准工作流程
```
完整设计流程:
1. Image Analyzer (图像分析)
   ↓
2. Terminate with Questions (需求澄清，如需要)
   ↓
3. Flux Kontext Max (主要设计生成)
   ↓
4. Image Analyzer (结果验证，可选)
   ↓
5. Flux Kontext Max (优化迭代，如需要)
   ↓
6. Terminate (任务完成)
```

### 快速设计流程
```
简化流程 (需求明确时):
1. Image Analyzer (快速分析)
   ↓
2. Flux Kontext Max (直接生成)
   ↓
3. Terminate (完成交付)
```

### 复杂设计流程
```
深度设计流程 (高要求项目):
1. Image Analyzer (深度分析)
   ↓
2. Terminate with Questions (多轮需求澄清)
   ↓
3. Poster Design Directions (设计方向探索)
   ↓
4. Flux Kontext Max (多方案生成)
   ↓
5. Image Analyzer (方案对比分析)
   ↓
6. Flux Kontext Max (精细化优化)
   ↓
7. Terminate (完成交付)
```

## 参数优化指南

### 不同场景的参数配置

#### 保守修改场景
```
参数设置:
- style_strength: 0.5-0.6
- quality: high
- aspect_ratio: 保持原图比例
- rescale: false

提示词策略:
- 强调保留原有特征
- 细微调整色彩和材质
- 保持整体设计语言
```

#### 创新设计场景
```
参数设置:
- style_strength: 0.8-0.9
- quality: ultra
- aspect_ratio: 根据需求调整
- rescale: 根据需求设置

提示词策略:
- 大胆的设计创新
- 新颖的材质和色彩
- 独特的视觉效果
```

#### 品牌升级场景
```
参数设置:
- style_strength: 0.7-0.8
- quality: high
- aspect_ratio: 9:16 (展示效果)
- rescale: false

提示词策略:
- 保持品牌核心元素
- 提升质感和档次
- 现代化设计语言
```

### 质量控制参数

#### 图像质量设置
```
质量等级选择:
- Ultra: 最终交付、重要展示
- High: 标准设计、专业需求
- Medium: 快速预览、概念验证

分辨率考虑:
- 高分辨率: 印刷、大尺寸展示
- 标准分辨率: 数字展示、网络使用
- 低分辨率: 快速预览、概念沟通
```

#### 一致性控制
```
种子值使用:
- 固定种子: 确保结果可重现
- 随机种子: 获得多样化选择
- 种子范围: 1-999999999

批量生成:
- 同一种子: 保持风格一致
- 相近种子: 获得相似变体
- 随机种子: 探索不同可能
```

## 错误处理和故障排除

### 常见错误类型
```
技术错误:
1. 图像URL无效或无法访问
2. 参数格式错误或超出范围
3. 提示词过长或包含无效字符
4. 网络连接问题或服务超时

设计错误:
1. 生成结果与预期差异过大
2. 产品特征丢失或变形严重
3. 色彩、材质表现不真实
4. 品牌元素缺失或错误
```

### 故障排除流程
```
问题诊断步骤:
1. 检查输入参数的正确性
2. 验证图像URL的有效性
3. 分析提示词的准确性
4. 评估期望与实际的差异

解决方案选择:
1. 参数调整: 修改相关参数设置
2. 提示词优化: 改进描述的准确性
3. 工具切换: 选择更适合的工具
4. 分步实现: 分解复杂需求
```

### 质量保证机制
```
预防措施:
1. 标准化参数模板
2. 提示词质量检查
3. 结果预期管理
4. 迭代优化控制

监控指标:
1. 成功率: 一次成功的比例
2. 满意度: 用户满意程度
3. 效率: 完成时间和迭代次数
4. 质量: 专业评估结果
```
