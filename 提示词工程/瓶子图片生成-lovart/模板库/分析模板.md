# 分析模板库

## Image Analyzer 分析模板

### 全面产品分析模板
```
分析重点：
1. 产品结构分析
   - 瓶身形状和比例
   - 瓶盖/泵头设计
   - 底部结构特征
   - 整体尺寸关系

2. 材质质感分析
   - 主体材质类型（玻璃/塑料/金属/陶瓷）
   - 表面处理工艺（光滑/磨砂/纹理/涂层）
   - 透明度特征（透明/半透明/不透明）
   - 质感表现效果

3. 色彩方案分析
   - 主色调识别
   - 辅助色彩搭配
   - 色彩饱和度和明度
   - 色彩心理效应

4. 品牌元素分析
   - 标志设计和位置
   - 字体选择和排版
   - 品牌色彩应用
   - 视觉识别一致性

5. 功能特征分析
   - 使用便利性
   - 储存功能性
   - 安全防护特征
   - 人体工程学设计

6. 美学风格分析
   - 整体设计风格
   - 视觉平衡感
   - 比例协调性
   - 文化内涵表达
```
