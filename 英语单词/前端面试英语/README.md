# 前端面试英语对话

本目录专门收集前端开发面试中常用的英文对话场景，帮助开发者提升英语面试能力。

## 目录结构

### 📝 [自我介绍](./自我介绍.md)
- 简洁版和详细版自我介绍模板
- 关键词汇和表达技巧
- 常见问题准备

### 💻 [技术问答](./技术问答.md)
- JavaScript、React、CSS等技术问题
- 标准回答模板和关键词汇
- 回答技巧和注意事项

### 🚀 [项目介绍](./项目介绍.md)
- STAR方法项目介绍框架
- 多个项目示例（电商、管理后台、移动端）
- 技术栈介绍和常见问题

### 🤝 [行为面试](./行为面试.md)
- 团队协作、问题解决、学习成长等场景
- STAR方法回答示例
- 关键词汇和表达句式

### 🤖 [AI工具使用](./AI工具使用.md)
- 教育邮箱账号申请和AI工具获取
- Cursor编辑器等AI开发工具的使用
- 日常开发工作流程中的AI集成
- 学习方法和团队协作场景

### ❓ [反向提问](./反向提问.md)
- 向面试官提问的技巧和策略
- 技术、团队、公司发展等方面的问题
- 提问时的表达方式和注意事项

## 使用建议

1. **熟练掌握基础表达** - 先掌握常用的面试词汇和句型
2. **模拟练习** - 多进行模拟面试练习，提高口语流利度
3. **准备充分** - 提前准备好自己项目的英文介绍
4. **保持自信** - 即使英语不够完美，也要保持自信和积极的态度
5. **持续改进** - 根据面试反馈不断完善自己的表达

## 学习重点

- 技术词汇的准确发音
- 逻辑清晰的表达结构
- 自然流畅的对话节奏
- 专业而友好的沟通方式
