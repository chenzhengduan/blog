**English:**
```
Interviewer: "What AI-powered development tools do you use in your workflow?"

Candidate: "I primarily use Cursor, which is an AI-first code editor built on VS Code. It integrates GPT-4 directly into the editing experience, allowing me to generate code, refactor existing code, and get explanations for complex logic. The inline suggestions and chat interface make it incredibly efficient for rapid prototyping and learning new technologies."
```

**中文对照：**
```
面试官："你在工作流程中使用哪些AI驱动的开发工具？"
