# 英语学习资料库

## 项目简介

这是一个按生活场景分类的英语学习资料库，包含单词、句子、文章等学习材料，每个材料都配有中文翻译和音标。本资料库包含以下24个生活场景分类，帮助您在实际情境中学习和运用英语。

## 场景分类说明

1. **日常生活** (`日常生活/单词.md`)
   - 基础生活词汇、家庭生活、日常活动等
   - 适合初学者和日常交流使用

2. **工作职场** (`工作职场/单词.md`)

4. **旅游出行** (`旅游出行/单词.md`)

5. **购物消费** (`购物消费/单词.md`)

## 使用说明

每个场景文件夹包含三类学习材料：
- **单词.md**: 该场景下的常用单词，包含音标和中文释义
- **句子.md**: 该场景下的常用句子和对话，包含中文翻译
- **文章.md**: 该场景下的短文和阅读材料，包含中文翻译

## 学习建议

1. 先学习单词，掌握基础词汇
2. 练习句子，理解语法结构和用法
3. 阅读文章，提高综合理解能力
4. 结合实际场景进行练习
