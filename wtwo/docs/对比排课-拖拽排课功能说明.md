# 对比排课 - 拖拽排课功能说明

## 📋 功能概述

在对比排课界面中，允许用户*- 📝 **简洁拖拽标签**：拖拽时显示紫色渐变的学员名称标签
- 🎯 **精确格子高亮**：实时高亮显示目标时间格子
  - 紫色渐变背景和边框
  - 旋转虚线边框动画
  - 脉冲缩放效果
  - 显示具体时间范围（开始-结束时间）
  - "📥 松开添加排课"弹跳提示
- ⏰ **分钟级精度**：支持精确到分钟的时间定位
- ✨ **流畅动画**：多种动画效果提升用户体验

### **数据自动填充**
- ✅ 学员信息：ID、姓名、头像、校区等
- ✅ 老师信息：ID、姓名、头像等
- ✅ 时间信息：日期、开始时间、结束时间（分钟级精度）
- ✅ 校区信息：自动继承学员所属校区课表**上，快速创建排课。

---

## 🎯 使用方法

### 1️⃣ **找到学员姓名**
在左侧学员视图的头部，可以看到学员的姓名和学号。

```
┌─────────────────────────────┐
│ 学员视图                     │
│ [刷新] 抦子 (SP175513) 🎯   │  ← 鼠标悬停会高亮，可以拖拽
│ [ℹ️] 19条                    │
└─────────────────────────────┘
```

### 2️⃣ **拖拽到老师课表**
- **鼠标悬停**到学员姓名区域，会出现淡紫色高亮背景和拖拽图标
- **按住鼠标左键**，从学员姓名开始拖动
- 拖动时会显示一个**简洁的学员名称标签**（📝 学员姓名）
- 拖动到右侧**老师课表**的任意时间位置
- **精确格子高亮**：当鼠标悬停在课表上时，会高亮显示具体的时间格子，显示：
  - 紫色渐变边框和背景
  - 具体的开始和结束时间（默认1小时）
  - "📥 松开添加排课"提示
  - 旋转虚线边框动画

### 3️⃣ **松开鼠标完成排课**
- 松开鼠标，系统自动：
  - 计算放置的精确日期和时间（分钟级精度）
  - 自动填充学员和老师信息
  - 打开排课弹框（与双击课表效果相同）
  - 默认课程时长为 **30分钟**（0.5小时）

## 🔧 技术实现

### **核心技术**
- **HTML5 Drag & Drop API**：原生拖拽支持
- **dataTransfer**：跨组件数据传递
- **pixelToTime()**：坐标转时间算法
- **事件通信**：`canlendar-dbclick-add-course` 事件

### **关键代码位置**

#### 1. 可拖拽的学员姓名区域
```vue
<!-- CompareTimeView.vue 模板 -->
<div class="student-details draggable-student"
     draggable="true"
     @dragstart="handleDragScheduleStart"
     @dragend="handleDragScheduleEnd"
     title="拖拽到老师课表进行排课">
  <span class="student-name">{{ props.assignList[0].Name }}</span>
  <span class="student-id">({{ props.assignList[0].Serial }})</span>
  <!-- 拖拽图标提示 -->
  <el-icon class="drag-icon">
    <svg><use xlink:href="#w2-paixu"></use></svg>
  </el-icon>
</div>
```

#### 2. 拖拽开始 - 创建精美的学员信息卡片
```javascript
// handleDragScheduleStart() - 创建拖拽幽灵图像
const dragImage = document.createElement('div')
dragImage.innerHTML = `
  <div class="drag-student-card">
    ${avatarHtml}  // 头像或首字母
    <div class="drag-student-info">
      <div>${studentInfo.Name}</div>
      <div>${studentInfo.Serial}</div>
    </div>
    <div class="drag-icon-wrapper">拖拽图标</div>
  </div>
  <div class="drag-hint-text">拖拽到老师课表排课</div>
`
// 设置紫色渐变背景
dragImage.style.background = 'linear-gradient(135deg, #667eea 0%, #764ba2 100%)'
```

#### 3. 老师课表接收
```vue
<!-- 课表网格添加拖拽监听 -->
<div class="schedule-grid"
     @dragover="handleScheduleDragOver"
     @drop="handleScheduleDrop">
```

#### 4. 放置处理逻辑
```javascript
// handleScheduleDrop() - 计算位置并触发排课
const dropPosition = pixelToTime(event.clientX, event.clientY)
const scheduleData = {
  date, startTime, endTime,
  object: studentInfo,
  teacher: teacherInfo,
  ...
}
emit('canlendar-dbclick-add-course', scheduleData)
```

## 📸 视觉效果说明

### **拖拽前（鼠标悬停）**
- 学员姓名区域显示淡紫色高亮背景
- 右侧出现拖拽图标（向右滑入动画）
- 光标变为抓手形状 (grab)

### **拖拽中**
- 跟随鼠标的紫色渐变卡片
  - 顶部：学员信息（头像+姓名+学号）
  - 底部：提示文字"拖拽到老师课表排课"

### **悬停在老师课表上**
- 整个课表区域高亮（淡紫色渐变背景+虚线边框）
- 中央显示"📥 松开鼠标以排课"（带脉冲动画）

### **放置后**
- 高亮消失

## ⚠️ 注意事项

### **限制条件**
- ✅ 只能从学员视图拖拽到老师视图
- ✅ 不支持老师拖拽到学员（业务逻辑限制）
- ✅ 拖拽到无效区域会被忽略

### **边界情况处理**
- 拖拽到过去日期：允许（由排课弹框处理警告）
- 拖拽到课表外：自动忽略
- 拖拽数据解析失败：静默忽略，不影响体验

### **兼容性**
- ✅ Chrome/Edge 支持良好
- ✅ Firefox 支持良好
- ✅ Safari 支持良好
- ❌ IE11 需要 polyfill（项目已不支持 IE）

## 📊 数据流向

```
学员视图 CompareTimeView
    ↓ (拖拽开始)
    设置 dataTransfer 数据
    ↓ (拖拽中)
老师视图 CompareTimeView
    ↓ (dragover 事件)
    显示高亮提示
    ↓ (drop 事件)
    解析拖拽数据 + 计算位置
    ↓ (emit 事件)
    emit('canlendar-dbclick-add-course', scheduleData)
    ↓
父组件 compareCanlendar.vue
    ↓
    handleCanlendarDbclickAddCourse()
    ↓
    emit('canlendar-dbclick-add-course', data)
    ↓
外层页面
    ↓
    打开排课弹框，自动填充数据
```
