# 青鱼养殖存活率数据生成器

## 功能说明

根据青鱼养殖存活率综合环境分析数据文档的接口，生成500条模拟数据并输出为Excel文件。

## 数据字段说明

| 序号 | 字段名称（中文） | 数据类型 | 说明 |
|------|----------------|----------|------|
| 1 | 养殖池编号 | 文本 | 唯一标识，格式如QY01、CY02等 |
| 2 | 投放时间 | 日期 | 青鱼投放日期，格式YYYY-MM-DD |
| 3 | 投放数量（尾） | 整数 | 初始投放数量，范围3000-8000尾 |
| 4 | 存活数量（尾） | 整数 | 当前存活数量，根据存活率计算 |
| 5 | 死亡原因 | 文本 | 可选：pH值异常、疾病、操作损伤、天敌等 |
| 6 | 养殖周期（天） | 整数 | 从投放到收获的天数，范围60-180天 |
| 7 | 检测时间 | 日期 | 数据检测日期，格式YYYY-MM-DD |
| 8 | 负责人 | 文本 | 负责养殖的人员姓名 |
| 9 | 水温（℃） | 浮点数 | 水体温度，范围20.0-28.0℃ |
| 10 | 溶氧（mg/L） | 浮点数 | 水体溶解氧含量，范围5.0-9.0mg/L |
| 11 | pH | 浮点数 | 水体酸碱度，范围6.5-8.0 |
| 12 | 氨氮（mg/L） | 浮点数 | 水体氨氮含量，范围0.05-0.3mg/L |
| 13 | 基础存活率（%） | 浮点数 | 自动计算，范围85%-98% |
| 14 | 综合存活率（%） | 浮点数 | 自动计算，考虑环境修正因子 |

## 算法说明

### 环境修正因子计算
```
环境修正因子 = 1 - (0.35 × |溶氧实测-溶氧最优| + 0.30 × |水温实测-水温最优| + 0.20 × |pH实测-pH最优| + 0.15 × |氨氮实测-氨氮最优|)
```

其中最优值：
- 溶氧最优值：6.0 mg/L
- 水温最优值：24.0 ℃
- pH最优值：7.0
- 氨氮最优值：0.1 mg/L

### 综合存活率计算
```
综合存活率 = 基础存活率 × 环境修正因子
```

## 使用方法

### 方法1：直接运行脚本
```bash
python generate_qingyu_data.py
```

### 方法2：在Python中调用
```python
from generate_qingyu_data import generate_qingyu_survival_data, create_excel_with_formatting

# 生成数据
df = generate_qingyu_survival_data(500)

# 创建Excel文件
create_excel_with_formatting(df, '青鱼养殖存活率数据.xlsx')
```

## 输出文件

脚本会生成一个名为 `青鱼养殖存活率综合环境分析数据.xlsx` 的Excel文件，包含：

- 500条青鱼养殖存活率数据
- 格式化的表格样式（标题行蓝色背景、白色字体）
- 居中对齐的数据
- 合适的列宽设置
- 完整的边框

## 数据特点

1. **真实性**：数据基于实际养殖环境参数范围生成
2. **逻辑性**：存活率与环境参数存在合理的关联关系
3. **多样性**：包含多种死亡原因、不同的养殖池编号和负责人
4. **时间跨度**：数据时间范围覆盖2023-2024年
5. **可重复性**：使用固定随机种子，确保结果可重复

## 注意事项

- 确保Python环境已安装pandas、numpy、openpyxl包
- 生成的Excel文件会保存在脚本运行目录下
- 如需修改数据量，可以在脚本中修改`num_records`参数
- 如需调整数据范围，可以修改脚本中的相应参数设置
