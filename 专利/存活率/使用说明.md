# 青鱼存活率数据生成器 - HTML版本使用说明

## 🎯 功能概述

这是一个基于HTML+JavaScript的青鱼存活率数据生成器，您可以直接在浏览器中打开使用，无需安装任何软件或依赖包。

## 🚀 使用方法

### 1. 打开页面
直接双击 `青鱼存活率数据生成器.html` 文件，或者用浏览器打开该文件。

### 2. 配置参数
- **数据条数**：输入要生成的数据条数（1-10000条）
- **文件名**：输入生成的Excel文件名（不需要输入.xlsx后缀）

### 3. 生成数据
点击"🚀 生成数据"按钮，系统将：
- 显示生成进度
- 自动下载Excel文件到您的下载文件夹
- 显示数据统计信息
- 显示数据预览

## 📊 生成的数据字段

| 序号 | 字段名称 | 说明 | 示例 |
|------|----------|------|------|
| 1 | 养殖池编号 | QY01-QY50格式 | QY01 |
| 2 | 投放时间 | 2024年内的随机日期 | 2024-05-01 |
| 3 | 投放数量（尾） | 3000-8000尾随机数 | 5000 |
| 4 | 存活数量（尾） | 根据基础存活率计算 | 4800 |
| 5 | 死亡原因 | 从预设原因中随机选择 | pH值异常 |
| 6 | 养殖周期（天） | 90-150天随机数 | 120 |
| 7 | 负责人 | 从预设负责人中随机选择 | 张三 |
| 8 | 水温（℃） | 正态分布生成 | 24.5 |
| 9 | 溶氧（mg/L） | 正态分布生成 | 7.8 |
| 10 | pH | 正态分布生成 | 7.2 |
| 11 | 氨氮（mg/L） | 正态分布生成 | 0.15 |
| 12 | 基础存活率（%） | (存活数量/投放数量) × 100% | 96.0 |
| 13 | 综合存活率（%） | 基础存活率 × 环境修正因子 | 81.6 |

## 🧮 算法说明

### 基础存活率计算
```
基础存活率 = (存活数量/投放数量) × 100%
```

### 环境修正因子计算
```
环境修正因子 = 1 - (0.35 × |溶氧实测-溶氧最优| + 0.30 × |水温实测-水温最优| + 0.20 × |pH实测-pH最优| + 0.15 × |氨氮实测-氨氮最优|)
```

其中环境参数最优值：
- 溶氧最优值：6.0 mg/L，权重：0.35
- 水温最优值：24.0 ℃，权重：0.30
- pH最优值：7.0，权重：0.20
- 氨氮最优值：0.1 mg/L，权重：0.15

### 综合存活率计算
```
综合存活率 = 基础存活率 × 环境修正因子
```

## ⚠️ 注意事项

1. 确保浏览器支持JavaScript
2. 确保浏览器允许下载文件
3. 生成大量数据时可能需要等待几秒钟

## 🎨 自定义配置

如需修改数据生成参数，可以编辑HTML文件中的JavaScript代码：

- `死亡原因列表`：修改可选的死亡原因
- `负责人列表`：修改负责人名单
- `环境最优值`：修改环境参数的最优值
- `环境权重`：修改环境参数的权重
- 数据范围：修改投放数量、养殖周期等参数的范围
