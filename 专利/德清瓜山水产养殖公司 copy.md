# 青鱼存活率数据

> 所属行业：农、林、牧、渔业  
> 数据来源：企业数据  
> 数据规模：800条（示例）  
> 更新频次：每月更新

# 应用场景
在水产养殖管理过程中，青鱼存活率数据是衡量养殖技术水平和环境管理成效的重要指标。通过对不同养殖池塘、批次的青鱼投放数量、存活数量等数据的持续采集和跟踪，企业能够精准分析各环节对成活率的影响，及时调整养殖策略，优化饲养管理流程。结合历史存活率数据，可用于预测养殖风险、制定科学的养殖计划、提升经济效益。青鱼存活率数据还可为品种改良、疾病防控、政策申报等提供数据支撑，助力企业实现高效、可持续发展。

## 算法规则说明
通过对企业日常采集统计的数据进行分析，数据清洗后，得到以下原始数据：

**基础数据**：养殖池编号、投放时间、投放数量、存活数量、死亡原因、养殖周期、负责人
**环境数据**：水温、溶氧、pH、氨氮


## 存活率智能算法说明

青鱼存活率在基础存活率计算的基础上，结合环境参数修正因子，采用如下综合算法：

### 1. 基础存活率计算
\[
基础存活率 = (存活数量/投放数量) × 100%
\]

### 2. 环境参数修正因子
考虑溶氧、水温、pH、氨氮四个关键环境参数对鱼类成活的影响，计算环境修正因子：

\[
环境修正因子 = 1 - (W_1 × |溶氧实测-溶氧最优| + W_2 × |水温实测-水温最优| + W_3 × |pH实测-pH最优| + W_4 × |氨氮实测-氨氮最优|)
\]

**算法参数详细说明**：
- **W₁（溶氧权重）= 0.35**：溶氧是鱼类呼吸的关键因素，权重最高
- **W₂（水温权重）= 0.30**：水温影响鱼类新陈代谢和免疫力
- **W₃（pH权重）= 0.20**：pH影响鱼类生理机能和水质稳定性
- **W₄（氨氮权重）= 0.15**：氨氮是主要的有毒物质指标

**最优值设定**：
- **溶氧最优值 = 6.0 mg/L**：青鱼适宜溶氧范围5.0-8.0mg/L，取中位偏上值
- **水温最优值 = 24.0 ℃**：青鱼最适生长温度22-26℃，取中位值
- **pH最优值 = 7.0**：中性环境最适合青鱼生长，pH范围6.5-7.5
- **氨氮最优值 = 0.1 mg/L**：安全阈值，超过0.2mg/L开始对鱼类产生毒害

**计算示例**（基于示例数据）：
```
环境修正因子 = 1 - (0.35×|7.8-6.0| + 0.30×|24.5-24.0| + 0.20×|7.2-7.0| + 0.15×|0.2-0.1|)
            = 1 - (0.35×1.8 + 0.30×0.5 + 0.20×0.2 + 0.15×0.1)
            = 1 - (0.63 + 0.15 + 0.04 + 0.015)
            = 1 - 0.835
            = 0.165
```

> **环境参数标准值说明**：以上最优值基于《淡水鱼养殖技术规范》(GB/T 18407.4)、《青鱼养殖技术规程》等行业标准，并结合德清瓜山水产养殖公司多年养殖实践经验确定。权重系数通过历史数据回归分析和专家评估确定，可根据不同养殖阶段、季节变化和养殖模式进行适当调整。
### 3. 综合存活率计算
最终存活率为基础存活率与环境修正因子的乘积：

\[
综合存活率 = 基础存活率 × 环境修正因子
\]

**完整计算示例**（基于示例数据）：
```
1. 基础存活率 = (4800÷5000) × 100% = 96.0%

2. 环境修正因子 = 1 - (0.35×1.8 + 0.30×0.5 + 0.20×0.2 + 0.15×0.1) = 0.165

3. 综合存活率 = 96.0% × 0.165 = 15.84%
```

> **注意**：示例中环境参数偏差较大导致修正因子很小，实际生产中应控制环境参数在最优范围内。环境修正因子可根据实际养殖经验和历史数据动态调整参数权重。

## 数据来源描述

青鱼存活率数据来源于德清瓜山水产养殖公司自主开发的数字化养殖管理平台。平台覆盖公司所有青鱼养殖池塘，由养殖负责人每日采集投放、存活等基础数据，并结合自动化水质监测设备实时上传环境参数。所有数据经过多重校验、清洗和脱敏处理，确保数据的准确性和安全性。平台支持数据的实时采集、日常汇总和周期性分析，数据更新频率高，覆盖面广，是企业养殖管理和科学决策的重要数据基础。

## 数据用途与场景

- **养殖管理优化**：通过存活率数据分析，及时调整养殖密度、饲料投放和管理措施，提高青鱼成活率和经济效益。
- **风险预警**：发现存活率异常波动，及时排查养殖风险，防止大规模死亡事件发生。
- **品种改良评估**：对不同批次、不同品种青鱼的存活率进行对比分析，指导品种选育和改良。
- **政策申报与补贴**：为政府补贴、项目申报等提供权威数据支撑。
- **企业决策支持**：为公司扩产、基地布局、技术改进等重大决策提供数据依据。
- **行业交流与标准制定**：为行业标准制定、技术交流、学术研究等提供数据基础。
